# 技术写作方法

总结自如何入门技术写作 (opens new window)

# 选主题

# 找素材

  • 经验分享
  • 学习笔记
  • 复盘反思
  • 主动查找

# 定主题

  • 分类

    • 技术类
    • 管理类
    • 软技能

  • 强调

    • 太大的主题不适合阅读，需要拆分
    • 将大标题，缩小为小主题

# 起标题

  • 标题是主题的直观体现，直观明了就好
  • 可以尝试采用吸睛标题，学习相关套路

# 列大纲

# 包含要素

  • 背景/提出问题
  • 是什么
  • 为什么
  • 怎么做
  • 结果怎么样

# 组织结构

  • 时序结构

    • 就是按照时间顺序来描述
    • 当介绍的内容和时间有关系时，考虑用时序结构

  • 总分总结构

    • 总分总结构就是先介绍整体(总论点)，再介绍局部(分论点)，最后再做个总结
    • 介绍局部时，可以采用 MECE 法则：相互独立，完全穷尽

  • 分论点结构

    • 分论点的结构，有并列结构，对比结构和递进结构

# 写内容

# 结论先行

  • 就是先写结论，然后再说理由
  • 可以让读者心里有个预设，知道下面会说什么

# 必要概念的解释

  • 简要介绍
  • 附上解释链接

# 使用数据和例子

  • 数据，能提升观点说服力
  • 例子，能帮助读者理解你的观点

# 合理使用图片

  • 有些时候，一图胜千言
  • 在文章中穿插些趣图，可以让内容显得轻松

# 语句通顺

  • 注意错别字
  • 拆分长句，简化理解

# 逻辑正确

  • 避免以偏盖全
  • 避免偷换概念

# 书写规范

  • 提升内容的可读性
