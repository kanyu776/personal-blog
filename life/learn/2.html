# 白话法律 42 讲

《白话法律 42 讲》 (opens new window)简要笔记

# 职场篇

# 劳动合同与社保

  • 和员工签订劳动合同时用人单位的法定责任，只要存在"事实劳动关系"，法律一样会认定为劳动关系进行并保护。

  • 如果公司超过一个月并不满一年还没和员工签订合同的话，应在此期限内向员工每月支付双倍工资。

  • 社保具有强制性，公司不给你办理社保属于违法行为。

  • 只要劳动关系存续期间，都应该依法缴纳社保，因此试用期间也应该为员工缴纳社保。

# 保密协议和竞业限制

  • 保守公司的商业秘密，是每一个工作者必备的职业素养和义务。不管是否有明文要求，不管是否有保密费，这都是一项法定义务。

  • 判定员工是否构成侵权泄密，并不在于协议约定了什么，而在于这部分内容是否确实属于公司的商业秘密。只要属于商业秘密，相关人员都具有保密义务，泄漏或私自使用都是违法的。

  • 在签订竞业协议时，约定期限不能超过 2 年，超出的部分无效。

  • 签订竞业协议后，在履行该义务时，可以得到相应的补偿金，要重点关注补偿金的多少和支付方式。

  • 竞业限制补偿金的发放时间由用人单位和劳动者共同协商，并在劳动合同或者竞业限制协议中作出约定。如果未作约定或者未协商的，应当按月支付。

  • 竞业期限内，公司即使没有支付补偿，协议也是有效的，应该先解除。

  • 当事人在劳动合同或者保密协议中约定了竞业限制和经济补偿，劳动合同解除或者终止后，因用人单位的原因导致三个月未支付经济补偿，劳动者可以请求法院或仲裁机构解除竞业限制协议。

  • 竞业协议如果包含“最终解释权归公司”属于霸王条款，无效。

# 商业信誉和商品声誉

  • 商业信誉，指企业在商业上获得的正面的社会评价，也就是商家的名声。

  • 商品声誉，指商品获得的的优良的社会评价，也就是商品的名声。

  • 损害商业信誉、商品声誉罪的判定依据如下：

    • 主观上是故意的

    • 适用对象不论单位还是个人

    • 存在捏造散布虚假信息的行为

    • 客观上有“严重情节”或造成“重大损失”

# 合同事项和意识

  • 合同主要内容是民事权利和义务关系的建立，改变或是终止，是平等主体之间达成一致意见所签订的协议，对于程序员外包而已，通常是劳务事情上的合同。

  • 如果你就职于某家公司，你和你所在公司的关系就不是平等关系，因为你们双方是管理和被管理的关系。所以你们之间的关系适用于《劳动合同法》，而不是《合同法》。

  • 即使合同没有约定付款时间，如果你已经履行合同义务，交互了产品，就可以单方面指定一个合同期限，要求甲方付款，如果对方存在拖欠事实，可以向有管辖权法院提起诉讼。

  • 签订合同需要注意 4 点：

    • 你和谁签的？

    • 付款期限是多久？方式是什么？交付条件是什么？

    • 违约事项有那些？违约后如何处理？

    • 管辖法院是哪里？

# 股权问题

  • 股权方面的纠纷，比较常见的有股权确认纠纷、股东出资纠纷、股权转让纠纷等。

  • 股东退出机制主要包括股东退出条件、股东退出的股权转让、股东退出的股权价值和股权的继承等问题。

  • 《公司法》有规定，股东之间可以相互转让全部或部分的股权，股东也可以向股东之外的人进行转让，但必须先通过书面的方式告知其他股东，否则不具有法律效力，其他固定具有优先受让权。

  • 股权问题并不高深，初创时期靠的主要是协议，要对股权所涉及的问题进行约定。正规流程，则是要写入公司章程并办理工商登记，这也是我们最后的依据。

# 期权问题

  • 期权，本质上是由公司或创始人书面承诺，在达成某项成就或者符合其他条件时，给某人一定比例的股权，让其可以成为公司的股东。

  • 在实际操作中，期权协议一般都是格式合同，相对应的权利义务，都是提前约定好，白纸黑字记录下来的。

  • 期权问题并没有配套的法律法规明文定义，所以只要不违法，期权更多体现为“意思自治”，也就是具有明显的合同属性，让双方根据需求，自己协商确定。

  • 核心员工获得股权的方式通常有两种，取决于最初的约定。

    • 一种是无偿得到股权；

    • 另一种是按协议约定的价格购买股权，当然，这个价格会远远低于市场价格。

  • 期权协议常见的行权期限是 4 年或 5 年，实现方式通常为分期行权，具体主要有这么两种方式。

    • 一种是“一年一变更”，每满一年，按约定条件，把期权按比例转为股权。

    • 一种是“两年一变更”，前两年按比例转换一次股权，剩下的满年限后再一次性全转完。

  • 在期权协议中，常见的集中关于离职问题的约定有：

    • 公司无偿收回

    • 公司以极低的价格收回

    • 公司按照签协议时约定的价格收回

    • 公司按照离职时的市场价收回

# 工伤

  • 工伤认定申请，应由单位在发生事故后的一个月内申请，如果单位不申请，员工可以在一年内提出申请。

  • 上下班途中发生的员工伤害，不管员工本人是步行、自行车、电动车、自行车还是汽车，只要是跟机动车发生的交通事故，并且在事故中不承担全部责任或主要责任，都属于工伤。

  • 对于工伤中“途中”的认定，既包括在正常工作时间的上下班的途中，也包括因加班或单位安排的其他事情外出的途中。另外，符合基本常识的合理时间段的迟到途中，也属于这个范围。

  • 工作中突然发病死亡，视同工伤；在工作中突然发病，但没有死亡，不能视同工伤；在工作中突然发病，经抢救 48 小时内（含 48 小时）死亡，视同工伤，超过 48 小时死亡，不能视同工伤。

# 裁员

  • 在你没过错的情况下，公司以任何理由提出解除劳动合同，都要给你经济补偿金。如果是违法解除，还需要支付经济赔偿金。

  • 经济补偿金的标准：

    • 每满一年，支付一个月工资；

    • 六个月以上不满一年的，按一年计算；

    • 不满六个月的，向员工支付半个月工资。

  • 经济赔偿金的标准

    • 经济补偿金 * 2

  • 关于 N+1：

    • 指员工没有过错，但是各种主、客观因素导致不能正常工作的情况，由公司单方面提出的解除合同。

    • N 就是年限，其实就是经济补偿金。+1 指的是，公司如果没有提前 30 天通知员工，就需要额外赔偿 1 个月的工资。

  • 关于 2N：

    • 指的是公司违法解除劳动关系的情况，这个时候需要赔偿，而非补偿，也就是经济补偿金 * 2。

    • 公司没有合法理由，也没有按照法定流程解除，以及产妇孕期辞退都属于这类。

  • 针对员工无任何过错，公司降薪来逼走员工，可以依法解除合同，申请仲裁。

# 名誉/肖像/著作权

  • 网上的图片、文章不能随便拿来就用，要尽量取得原作者的同意，注明作者以及出处；来历不明的东西尽量不用。如果遇到版权方维权，要及时补救或删除。

  • 明星或球星照，不经授权不要商用，也不能用于公司的广告宣传。否则一旦你的公司发展壮大，就可能会有人找人门来要求赔偿。

  • 不在网上发布别人的负面信息，不诋毁、诽谤别人，也不要私自传播别人的隐私，否则都可能要承担相应的法律责任。

# 工资/加班/休假

# 工资

  • 劳动报酬分为三个部分：

    • 一是工资，包括基本工资、奖金、津贴、补贴等；

    • 二是实物报酬，包括公司发的各种物品或是提供的有价服务等；

    • 三是社会保险，主要就是我们所说的“五险”。

  • 试用期的工资，不应该低于本公司相同岗位最低档工资或劳动合同约定工资的 80%，也不应该低于公司所在地的最低工资标准。

# 加班

  • 一般情况下，员工不同意，公司并不能强行要求员工加班，更不能认为，员工不同意加班就是不服从公司安排。

  • 如果公司安排加班，应该按照国家规定，向员工支付加班费，而加班时间不同加班费的支付标准不同。

  • 加班限制性规定：

    • 每天安排的加班时间，不能超过 1 小时；

    • 如果因为特殊原因需要延长，在保障员工身体健康的条件下，每天不超过 3 小时；

    • 每月安排的加班总时间，不能超过 36 小时；

    • 公司不能安排未成年人、怀孕女员工以及哺乳期未满一年的女员工加班。

  • 加班费的规定：

    • 如果是普通工作日加班，公司应该 1.5 倍工资作为报酬；

    • 如果是休息日加班，公司应该发 2 倍工资作为报酬，或者可以选择补休；

    • 如果是法定节假日加班，公司应该发 3 倍工资作为报酬，不能选择调休来补偿。

# 休假

  • 年休假，也就是我们通常所说的年假，是指员工每年可以享受的，保留工作和工资的一段连续休息时间。

  • 我国实行的是“带薪年休假”制度，也就是员工连续工作 1 年以上的，可以享受带薪休年假。

# 受贿/行贿

  1. 非国家工作人员受贿罪，指非国家工作人员，利用职务之便，索取或者非法收取别人的大额财务，帮行贿人谋取利益的行为。

  2. 不管你帮别人得到的利益合法还是非法，应得还是不应得，都叫做谋取利益。显然，只要收了贿赂的财物，就有犯罪的风险。

  3. 如果收取费用合规，员工可以先交上公司，再由公司以奖金形式还给自己。

# 技术篇

# VPN 搭建

  1. 根据目前的法律来看，私自搭建“VPN”并且售卖的行为，明显是违法的。

  2. 技术中立原则强调的是技术本身无罪，但是开发者和使用者是否有罪，要看具体情况界定。

  3. 平台确责原则强调的是，平台被利用犯罪，平台经营者也需要承担一定的责任，并且这一原则有明显强化趋势。

  4. 程序员作为公司的开发人员，如果对公司业务“不知道也不应当知道”，就不用承担刑事责任；如果明知故犯，则可能会构成共犯，同样承担刑事责任。

# 系统漏洞

  1. 国家事务、国防建设、尖端科学技术领域，这三个领域的计算机系统碰不得，侵入即犯罪。——非法侵入计算机信息系统罪。

  2. 其他领域，入侵系统或者获取数据，情节严重的会被判三年。——非法获取计算机信息系统数据。

  3. 情节严重的门槛非常低，比如非法获利 5000 元就是犯罪了。

# 爬虫与信息

  1. 怎样获取数据，获取的数据是什么范围，以及把数据用来做什么，是爬虫与信息合法性问题的关键。

  2. 严格遵守"Robots"协议，不越权，获取公开可查询的非隐私信息不违法，其他类型的可能违法。

  3. 未经搜权，强行突破反爬措施，导致被爬网站运行受到严重影响，违法。

  4. 第三方应用获取信息要注意"三重关卡"。

# 非法集资

  1. 非法集资本身不是一个罪名，一般会涉及非法吸收公众存款罪和集资诈骗罪。

  2. 非法吸收公众存款，也就是俗称的"非吸"。个人非吸的数额达到 20 万以上，或是单位非吸的数额在 100 万以上，就构成了犯罪。

  3. 个人集资诈骗达到 10 万，或是单位集资诈骗达到 50W，就构成了犯罪。

  4. 非吸的最高刑期是 10 年，而集资诈骗罪最高为无期徒刑。
