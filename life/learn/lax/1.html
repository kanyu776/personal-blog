相关专栏的笔记整理，比较零散简要。

# 1.自信是恋爱成功的第一关键

# 自信

  • 自信可以为约等于你的内心状态。
  • 不自信来源于价值对比。
  • 不自信的人通常会索取价值。

# 提升自信的路径

  • 做符合大众价值观的事情（场景自信）
  • 扭带关系（场景自信）
  • 特殊技能（场景自信）
  • 角色扮演（场景自信）

# 2.什么是吸引？价值和传递价值

  1. 价值产生吸引，应该学会传递价值。

  2. 能产生好情绪的东西都是有价值的。

# 如何传递价值：

  • 潜沟通
  • 行为线索

# 价值的分类

  • 硬价值（30%）

    • 生存价值

      • 金钱
      • 社会地位
      • 前途

    • 繁殖价值

      • 穿搭
      • 形象
      • 体能

  • 软价值（70%）

    • 情绪提供力

      • 推拉
      • 打压
      • 幽默

    • 价值认同感

      • 嫉妒心/虚荣心
      • 剥夺认同感
      • 投资陷阱/假性失格

    • 雄性领袖特质

      • 意图清晰
      • 控制力
      • 理所当然感
      • 自我娱乐

    • 性格价值

      • 幽默
      • 高情商

    • 品质和沉淀

      • 修养
      • 积累

# 3.释放吸引的三种信号

# 吸引力的表现形式

  • 硬价值——任何属于你的外在价值：长相身高，经济实力，地位权利

  • 软价值——任何能传递出你具有潜在生存和繁衍价值的行为都是软价值

# 吸引力传递的 3 种信号

  • 视觉信号

    • 外在形象

  • 行为信号

    • 行为印象
    • 性格教养

  • 社交信号

    • 他人对你的肯定和认同
    • 他人对你的评价和顺承

# 好的行为线索（基础模板）

  • 全身放松，泰然自若
  • 挺直腰板，不要驼背
  • 说话的时候，语速要慢
  • 眼神坚定，不要漂乎不定
  • 正常的社交礼仪

  1. 喜欢是情绪上的，而非逻辑上的。

  2. 女生判断一个男人的吸引力，在于他的行为线索。

  3. 女人其实喜欢性爱。

# 女人对待男人的 2 种态度

  • 情人

# 吸引女性的方法

    • 眼神
    • 语气
    • 语调

  • 专注于情绪（提供情绪价值）

    • 增加开心的感觉

# 5.吸引力

# 吸引的本质——价值

  • 价值等于吸引。

  • 价值形式：

    • 硬价值

    • 软价值

      • 核心自信
      • 情绪价值
      • 雄性领袖线索

# 男性价值的参考指标——行为线索

  • 对于女人来说，判断一个男人的吸引力，主要源于潜沟通中的男性表现出来的行为线索。

  • 行为线索

    • 潜沟通

    • 肯定感

  • 是否评论动态
  • 是否想见面

# 如何有效提高吸引力

  • 硬价值

    • 形象

  • 软价值

    • 情绪价值的提供（聊天技巧）
    • 雄性领袖特质的展现

# 6.一致性

  1. 在社交中是谁在主导这段对话，又是谁在迎合对话。
  2. 男人是提供情绪和能量的一方，是主导的一方。
  3. 真实地表达自己，不迎合动摇，有控制力。

# 7.在状态

  1. 在状态，能够传递价值。
  2. 在状态的人，不依赖任何话术技巧也能影响他人。

# 什么是在状态

  1. 满足感
  2. 理所当然感
  3. 感觉自己有权力做任何事
  4. 感觉是社交场合的中心

# 在状态的感受

  1. 心态平静淡定
  2. 情绪高扬

# 状态类别

  • 正向：主动，强大，有能量
  • 反向：被动，懦弱，无能量

# 状态的决定因素

  1. 世俗价值观（软件）
  2. 角色扮演（软件）
  3. 关系（软件）
  4. 技术（软件）
  5. 核心自信（硬件）

# 影响状态好坏的因素

  • 条件自信

    • 世俗价值观
    • 关系
    • 角色
    • 技能

  • 核心自信

    • 特点

      • 不依赖任何外在条件
      • 能扭转因果关系

    • 提升方法

      • 不断地充实自己
      • 自娱自乐的精神
      • 真实地表达自己

# 8.四心态

# 丰富心态

  1. 与之相对的是稀缺心态。
  2. 越不在乎的人越有掌控力。

# 魅力心态

  1. 直接假设是被喜欢的。

# 真实心态

  1. 好奇和渴望。
  2. 真实与坦白。
  3. 不要为自己的天性感到羞愧。

# 价值心态

  1. 需求和价值是平等的。
  2. 我在升值，你在贬值。
  3. 对于女生来说，是逐渐下降的。

  • 首先你要平常化你的目标，不要仰望她。
  • 通过提升外形来提升自己的优越感。
  • 降低预期。
  • 提高筛选标准和精神层次。
  • 学会怀疑自己。
  • 把约会当成玩。

# 场景自信和核心自信

  • 自信源于价值。
  • 一个男人是否优秀是次要的，接纳自己，认可自己才是更重要的。

# 10.取悦自己，而不是女人

  1. 取悦自己，而不是女人。

  3. 不要设置太高的表达门槛（多厉害/有趣）。

  2. 无论我说什么，都是有趣的，纯粹是因为从我口中说出来的。

  • 让她进入你的生活中
  • 让她感受挑战
  • 让她适应你

# 15.识别女生聊天信号模型

# 识别信号灯

  • 判断红灯

    • 是否高冷
    • 是否简短
    • 是否提供话题
    • 很逻辑不带情绪
    • 敷衍程度

  • 判断黄灯

    • 是否有废物测试（一致性测试）

  • 判断绿灯

    • 是否主动问问题
    • 是否有正面情绪
    • 回复速度和频率
    • 表情和语气词

# 应对方法

  • 遇到红灯

    • 及时止损
    • 表明态度
    • 推开对方
    • 展示价值

  • 遇到黄灯

    • 推回废物测试（问题）
    • 通过废物测试
    • 忽略废物测试

  • 遇到绿灯

    • 假定追逐（对自己）
    • 表达兴趣（对女生）
    • 进行赞美（对女生）

# 26.逆袭

# 框架参照模板

  1. 我是一个有自信有勇气有能力的男人。
  2. 我要真诚快乐的谈恋爱，我永远把工作和成长放在第一位。
  3. 我讨厌被怀疑和强迫，我也讨厌怀疑和强迫别人。
  4. 我努力进行外在建设，举止得体，心态健康，生活精彩。
  5. 我要努力成为更好的自己。
  6. 我可以拥有任何美丽善良有爱的女人。
  7. 我会珍惜和把握住这样的女人。
  8. 我是女人的奖品。

# 建立属于自己的框架模板

  • 硬框架（不等于好框架）

    • 维护的是自身利益
    • 吸引力高，可得性低

  • 软框架（不等于坏框架）

    • 维护的是他人利益
    • 可得性高，吸引力低

  • 好框架

    • 软硬皆有
    • 动态调整框架软硬

  • 坏框架

    • 只有其一

# 如何吵架

  • 炒架时，用硬框架
  • 冷静后，用软框架

# 如何屌丝逆袭

  1. 靠软价值/内在价值
  2. 靠框架和态度
  3. 靠框架的软硬调整
  4. 忽略/弱化/打压对方硬价值（扬长避短）
