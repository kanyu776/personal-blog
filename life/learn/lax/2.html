# 直男改造手册

# 1.两性吸引本质

# 高质量的关系构成

  • 吸引力
  • 可得性
  • 投入度

# 吸引力构成

  • 心动指数（90%）——女儿对父亲般的崇拜，由价值和能量构成

  • 心疼指数（10%）——母亲对孩子般的疼爱，由暴露敏感面触发

# 如何构建吸引力

  • 视觉形象
  • 行为习惯
  • 社交认证

# 直男需要注意的

  • 自信是恋爱成功的第一关键。
  • 不要仰望女生，不存在女神之说。

# 2.亲密关系模型

  1. 爱是給予，不是索取，是满心的欢喜，是无需求。

  2. 你跟女生的聊天互动会得到 2 种反馈：

    • 无感窗口 -> 进行高价值展示

    • 好感窗口 -> 进行服从性测试

      • 通过，做关系升级
      • 不通过，继续高价值展示

# 恋爱六阶段

  1. 认识阶段——通过各种途径认识女生，结识女生。
  2. 了解阶段——通过基本展示和沟通，建立安全感，熟悉感，舒适感。
  3. 吸引阶段——通过高价值展示，建立我们对女人的吸引力。
  4. 示好阶段——在确定一定的吸引力后，给予女人适当的可得性。
  5. 诱惑阶段——通过语言和行为释放暧昧信号，升级关系，构建亲密感，为爱“鼓掌”。
  6. 交往阶段——建立女人对我们的依赖感和归属感。

# 记住三点

  1. 所有和女生的开场，都是无需求。
  2. 所有自身的高价值展示，都是对方所不具备的。
  3. 恋爱六个阶段，只有示好和暧昧阶段可以调换，其他阶段不可以。

# 3.良好的第一印象

  1. 第一次见面，形象应该用心打理，要干净、得体、阳光和帅气。

  2. 你在和女生沟通时，存在著名的 7/38/55 定律：你和女生在沟通中，重点不在于聊天的内容，而在于你是谁。

# 7/38/55 定律

  • 7 代表内容
  • 38 代表语速/语调/语音
  • 55 眼神/表情/姿体动作/社交认证

# 4.自我建设基础

  1. 自身建设从简到难，分别为:形象管理、时间管理、情绪管理和技能管理。

  2. 健身是提升形象非常有效的方法，也是性价比最高的方式。

  3. 学会完善自己的性格，大男子主义和极端情绪化都是恋爱杀手。

  4. 自我建设要从学会尊重和同理心开始，培养自己的共情能力。

  5. 拓展技能对自己个人是很大的加分项，人是需要提升和努力的。

# 5.学会展示自我

  1. 普通人的展示平台主要是微信和微博，一般主要是微信。

  2. 朋友圈（微博）建设不在于花多少钱，而是你要有一双发现美的眼睛。

  3. 朋友圈（微博）展示内容

    • 生活/兴趣爱好
    • 工作状态
    • 生活品质
    • 个人品位
    • 社交认证
    • 展示情趣价值
    • 展示生存价值
    • 展示技能

# 6.框架筛选模型

  1. 框架就是你筛选女生的标准，和女生相处的原则。

  2. 框架相当于游戏的难度，框架高代表吸引力很高，但是可得性较低；框架低代表吸引力很低，可得性很高。

  3. 女人最终选择的男人，是能在吸引力和可得性之间求取到平衡的男人。

  4. 可得性建立在吸引力之上。你对女生有吸引力时的给予，叫赋予可得性；反之，叫暴露需求感。

  5. 暴露需求感是男女交往的大忌，特别是针对于男性，是一般舔狗的主要特征

  6. 跪舔女生是因为在与女生相处过程中丢失了自己的框架，服从了女生的框架，丢失了吸引力。

  7. 框架的碰撞在日常生活中是随时发生的。防止跪舔的方法是建立自己的框架，并且在交往中坚持自己的框架。

  8. 尊重女生是发生在男强女弱的情况下，不是女强男弱；女强男弱的情况下谈尊重女生，那就是窝囊废。

# 7.释放社交信号

  1. 通过与其他女性的正常合照，释放预选和社交认证，突显自身价值。

  2. 朋友圈的照片中，可以尝试三人以上的合照，画面应该没有暧昧。

# 8.正确恋爱顺序

  1. 提升外在形象（最直接明显的改变方式）

  2. 做好展示面（通过朋友圈）

  3. 认识大量女生（30 个以上）

  4. 应用框架筛选

  5. 和女生通过聊天和约会，做一些事情升级关系

# 9.如何冷读女生

  1. 冷读，就是通过观察女生（或朋友圈），根据自己的认知和经验，做出一个猜测并开启话题，调出女生内心的想法，再进行附合，认同和引导，最终达到和女生拉升关系。

  2. 冷读一定要用“我以为，我觉得”来开头。

  3. 冷读只能作为聊天的点缀，不能作为聊天的主体。

  4. 冷读的目的不是宣布一个“结论”，而是以我们有这种感觉为诱饵，引导女生说出更多的信息。

  5. 冷读最关键的点不在于我们说了多少，而在于我们听到了多少。

# 冷读方法

  • 观察冷读（一般用在了解阶段）
  • 聆听对方（让女生有种被重视的感觉）
  • 营造交集（基于女生所聊，找共同点深入）

# 10.破解女生矜持

  1. 一个女生就算心理再喜欢一个男生，也不会追求和表白，这便是女生的矜持。

  2. 矜持是女性恋爱的语言，它让前期关系更加浪漫，让后期关系更加稳固。

# 女生为什么会矜持

由于进化生物学天职决定

  • 男性生育成本很低，因此可以走量
  • 女生生育成本很高，因为需要走质
  • 所以男性会贪多求快，更为花心
  • 所以女性要识别筛选，更为谨慎

# 女生矜持的好处

  • 可以延长男性追求的过程，确保没被骗。
  • 通过增加男性投入度，让男性离开成本变大。
  • 避开承担关系推进责任，避免被指责为荡妇。

# 11.女生窗口识别

# 窗口识别

  • 好感窗口

    • 女生在一切语言，行为，身体和兴趣爱好上愿意配合你

    • 好感类型

      • 主动型——主动做促进交往的事情
      • 被动型——对你的主动保持好的响应性

  • 无感窗口

    • 不会主动询问你的事情
    • 对你表达的事情没有表现出兴趣
    • 和你聊天过程中的表情和情绪不够
    • 跟你保持一个合适的社交距离

  • 反感窗口

    • 女生在一切语言，行为，身体和兴趣爱好上都不愿意配合你

# 约会中的好感窗口识别

身体 5 个部位（面，胸，胯，膝，脚尖）

  • 全部朝向你，即是好感窗口
  • 部分朝向你，即是无感窗口
  • 都没朝向你，即是反感窗口

# 窗口识别的注意点

  1. 识别到好感窗口后，要及时做服从性测试。

  2. 聊天中不要一直问女生，这是索取价值的行为，会让女生反感。

  3. 问女生时，应该将疑问句变成陈述句或描述句，由提问变成引导。

# 12.女生约会秘籍

  1. 约会好感定律：凡是愿意单独出来见你的女生，都是对你有好感。

  2. 牵手定律：普通朋友是不会和你牵手的，女生只有对亲密关系的朋友才会牵手。

  3. 关系升级定律：牵手=>拥抱=>接吻=>搜身=>做爱。

  4. 第一次约会最重要的是：不丢分。

# 整体约会流程

  • 第一次，建立好感（奶茶店/咖啡店）
  • 第二次，牵手测试（吃饭/电影/徒步）
  • 第三次，接吻测试（游乐场/摩天轮/景点）
  • 第四次，私密测试（带入私密空间）

# 13.聊天你要会撩

  1. 聊什么不重要，你是谁才重要。
  2. 女生愿不愿意回你消息，取决于你是否有吸引力。
  3. 让自己的生活变得有趣，聊天才会有话聊。

# 聊天话题

  • 美食/星座/旅行/宠物
  • 朋友圈筛选

# 延展话题的技巧

  • 描述状态
  • 表达感觉
  • 适时表情

# 聊天注意事项

  • 做好价值展示

    • 无意展示
    • 不装逼

  • 思想层次展示

    • 敢于表达状态和感受
    • 敢于表达真实想法和自己

  • 保持自己的框架

    • 不跪舔
    • 保持对称/平等
    • 不害怕想法被否定

# 14.自身价值展示

真正的高价值是从处理问题中产生的，焦点在问题上，价值体现在人身上。

# 个人价值的分布区间

  • 高调区间

    • 最容易吸引到女生
    • 短期属性

  • 低调区间

    • 适合长期关系
    • 感情和婚姻会比较稳定

  • 中庸区间

    • 可以吸引到普通女生

  • 装逼区间

    • 可以造成吸引，但容易一致性断裂

# 关于打压

  1. 打压，是一种在掩饰需求感的基础上，对女生表达在乎的技巧。
  2. 打压，是高价值男生在和女生相处时的态度，只有高价值的男生才会对女生进行打压和推拉技巧。
  3. 打压的目的，是为了掩饰需求感和忽视女生存在的价值，防止自己无意识的跪舔女生（神）。

# 15.付出和吸引力

  1. 付出，一定要在你确认对女生有吸引力的前提下。

  2. 付出，是两个人确认关系之后的事情，在此之前你需要做的是不断提升自己的价值。

  3. 追女生的时候，一定要做好两件事：

    • 努力提升自己
    • 引导对方投入

  4. 人生的必修课是绽放自己，而不是讨好他人。

  5. 女人是在吸引力和可得性之中寻找对象。

# 16.什么时候表白

  1. 现实中表白失败的本质原因在于，只在乎了表白的形式，忽视了两人关系的程度和女生的感受。

  2. 表白是赋予可得性最高的行为，它是吸引力胜利的钟声，而不是冲锋的号角。

  3. 表白只能是建立好感和吸引力后的行为。

  4. 表白之后有两种结果，表白成功进入亲密关系，表白失败关系倒退或破裂。

  5. 如果想要表白，一定要先做测试，比如：

    • “假如你做我女朋友的话...”
    • “如果...”
    • 要传递正向的情绪价值，传递好情绪

# 17.感情如何挽回

作为男人，你要明白，被分手是一件再正常不过的事情了。

# 被分手的常见原因

  • 木讷，太闷，不会调情，不够坏。
  • 太风骚，太坏，太浪。
  • 太情绪化，缺乏安全感，占有欲强。
  • 不够上进，不积极，不努力。

# 被分手可以采取的措施

  • 一定要冷静下来，不要联系她

    • 沟通建立在双方都有意愿的基础上
    • 一方有意，一方无意，沟通也是徒劳

  • 学会控制好自己，不要损陷自己

  • 重新找回自己，做好自身建设

# 18.常见恋爱误区

  1. 在认识阶段，没有经过了解，吸引和示好阶段，直接进入诱惑阶段（必死无疑）。

  2. 在吸引阶段之前进行示好，暴露需求，赋予女生可得性和投入度（笨蛋行为）。

  3. 从吸引阶段直接跳到诱惑阶段，缺失了了解阶段和示好阶段（大概率失败）。

  4. 不敢进入诱惑阶段，而被放置在了友谊区（友谊区很难出来）。
