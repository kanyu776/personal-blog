# 恋爱学理论

# 1.恋爱六阶段模型

  1. 认识阶段——通过各种途径认识女生，结识女生。

  2. 了解阶段——通过基本展示和沟通，建立安全感，熟悉感，舒适感。

  3. 吸引阶段——通过高价值展示，建立我们对女人的吸引力。

  4. 示好阶段——在确定一定的吸引力后，给予女人适当的可得性。

  5. 诱惑阶段——通过语言和行为释放暧昧信号，升级关系，构建亲密感，为爱“鼓掌”。

  6. 交往阶段——建立女人对我们的依赖感和归属感。

# 2.男女思维差异

  1. 女性思维有 2 大主要的特征：情绪主导和关注细节。

  2. 女人都是情绪动物，甚至在很多时候，她们的情绪盖过理智。

  3. 在面对同一件事情的时候，男人会去阐述事实，而女人更关注感受。

# 3.女性择偶偏好

  1. 生存和繁殖，指导着我们的求偶行为。

  2. 生存价值，就是我们在这个世界上的生存能力的指标；繁殖价值，是衡量我们后代基因是否优良的指标。

  3. 两性择偶，就是对生存价值和繁衍价值的选择。只是比较而言，男人更看重女人的繁衍价值，女人更看重男人的生存价值。

  4. 男人播种，女人选种，是一种生物本能。

  5. 正因为女人本身的选种机制，她们必然有一个挑剔和筛选的过程。所以，女人跟男人的约会其实就是一种筛选过程。

# 4.女人的矜持

  1. 在两性择偶上，女人择偶会保质，男人择偶会保量。

  2. 女人对生育成本的考虑，使得矜持成为选择。

  3. 男人的主动是进攻，女人的主动是配合。

# 矜持的原因

  • 做出更理智的判断

    • 浅层原因

      • 确认有价值的男性
      • 确认自己是否有投入繁殖价值的必要

    • 深层原因

      • 利于良好的生存和繁衍

  • 增加男性的投入

    • 浅层原因

      • 增加了关系前期得到的男性投资
      • 有助于关系后期的稳定性
      • 得到更大的安全感
      • 使男性较难轻易放弃

    • 深层原因

      • 利于良好的生存和繁衍

  • 不必承担推进和深入关系的责任

# 应对矜持的方法

  1. 不要管女人嘴上怎么说，而是要去观察窗口。
  2. 女人的窗口，才是男人行动最准确的信号。
  3. 男人识别到好感窗口后，要主动发起进攻。

# 5.男人的主动性

  1. 一切主动都是建立在吸引的基础上。

  2. 男人需要主动，必须主动。

  3. 要对女生的窗口保持敏感和关注。

# 男人主动的方法

  • 胆大
  • 心细
  • 脸皮厚

# 6.吸引力

  1. 所谓吸引，不过是女生对我们生存价值和繁衍价值的一种判断。

  2. 当我们对女人产生足够的吸引时，女人自己也没法意识到自己所表现出来的一切言行和好感都源自本能。

  3. 当一个男人的生存和繁衍价值越高，他就越受女人欢迎。

  4. 吸引是人的一种本能反应，而非理性选择。

  5. 吸引的三种信号：视觉信号、行为信号和社交信号。

# 7.可得性

# 什么是可得性

  1. 男人愿意在女性抚育孩子的过程中，保证她足够的生存能力，这个意愿就是男人的可得性。

  2. 可得性代表你有多大的可能，愿意持续地把你的价值提供给女人。这个价值既包括繁殖价值，也包括生存价值。

# 可得性和需求感的区别

  1. 没有吸引，就是需求感。有了吸引，就是可得性。

  2. 可得性是建立在吸引的基础上。

  3. 没有吸引，它就会变成一种需求感，而需求感是不会获得女人喜欢的。

# 吸引与可得性

  1. 我们需要恰当的吸引，再具备良好的可得性，这样女人才会感受到一种平衡和舒适感，才会想要跟我们长期地在一起。

  2. 最终决定女生选择的，就是吸引力和可得性的均值。

# 8.投入度

  1. 投资要平衡，需要引导女生为你投资。

  2. 投入的东西要恰当，投入的方式要贴心。

  3. 自己投入得好，自然也能带动女生去做。

# 9.需求感

  1. 需求感实际上就是你对女生的欲望，对女生的亲近需求和繁殖价值的索取。

  2. 有吸引之前，就对女生做示好的事情就叫做暴露需求感，是会让我们减分的。

  3. 有吸引之后，同样的示好行为叫做给予可得性，这才会让我们加分。

  4. 在吸引之前，我们要做的就是无需求感。

  5. 女人天生是隐藏需求感的好手，我们要想在关系中占领高地，就要懂得隐藏需求感。

  6. 隐藏需求感，可以让我们保留自己在女生心目中的价值。

  7. 隐藏需求感，要把握好度，不要让女生以为我们对她没兴趣，因此要在吸引建立后要适当地给予可得性。

  8. 千万要明白一个道理，你把自己当成什么人，别人就会把你当成什么人对待.

# 隐藏需求感的三种方法

  1. 间接邀约
  2. 有偿帮助
  3. 善用巧合

# 要想获得自我价值感，我们需要想明白三件事

  • 自己的能力是有限的
  • 我们的自我价值感来源于自己的内心，而不是由别人决定的
  • 不要被一时的得失所左右

# 在遇事的时候，如何保持稳定的自我价值感

  • 关注自己
  • 列一个成就清单
  • 与不合理信念辩论

# 如何建立自信

  • 多做自己擅长的事情
  • 不要过于敏感
  • 用外在影响内在

  • 增长自己的见识

# 13.框架

  1. 简单来说，框架就是你与人交道和社交恋爱时的准则，属于你自己的“社交宪法”。

  2. 你是一个什么样的男人，是根据你的框架来决定的。

  3. 框架和一个人的价值无关，而是和能量有关。

# 硬框架和软框架

  1. 框架有软硬之分，硬框架是对待事情时以自己的感受为主，软框架是以对方的感受为主。

  2. 当你选择用硬框架保持自我的时候，你的吸引力不会丢失，但是你的可得性会丢失。

  3. 当你用软框架对待女生的时候，你的可得性会很高，但是吸引力会随之下降。

  4. 软硬框架就像阴阳两极，缺失任何一方，你的框架都无法平衡。

  5. 框架软硬平衡，才能让吸引力和可得性也变得平衡，而后者的平衡才能让关系健康平稳的发展。

  6. 软硬框架没有好坏之分，他们之间的关系是相辅相成的。

  7. 最好的框架是软硬平衡，让你的吸引力和可得性维持在一个良好的区间里。

# 框架推拉

  1. 男人必须要有自己的框架，在发生框架碰撞时，要能守住自己的框架，而方法就是推拉。

  2. 框架的推拉，一定要先推后拉。

  3. 框架推造成的负作用，可以通过拉来补回。

  4. 框架的拉，一定要在女生生完气了，没再抱预期后再做（这样才能让女生的情绪产生波动，制造出意外的感觉，浪漫的感觉）。

  5. 通过框架推拉，能将女生本来觉得理所应当的事情变得具有奖赏性，能保持住吸引力的同时又具备可得性。

  6. 一个人的能量高低不是恒定的，我们要根据所处环境的变化去切换自己的能量。
