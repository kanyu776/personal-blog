# 《驱散社交焦虑的阴霾》

《驱散社交焦虑的阴霾》 (opens new window)是一本简洁和实用的自助手册，针对社交焦虑有着非常清晰有效的策略方案指导，下面是我针对实际有效的方法论章节的提取和整理，主要集中在6~15章。

# 第六章 克服社交焦虑的三个步骤

  • 了解自己的焦虑模式

  • 改变自己对引发焦虑的情境的想法

  • 改变自己的焦虑行为

# 第七章 战胜焦虑的想法

  • 负面想法是社交焦虑的核心

  • 将注意力转移到他人身上

  • 识别情境中的焦虑想法，信念和假设

  • 质疑焦虑背后的信念和假设

  • 确立应对之法

# 第八章 控制自己的身体

  • 减焦策略

    • 接受

    • 转移注意力

    • 积极应对

    • 放松

  • 放松策略

    • 放松呼吸法

    • 深度既然

    • 想象放松法

# 第九章 改变行为，直面恐惧

  • 回避行为会维持焦虑

    • 直面恐惧

  • 为直面恐惧制定任务

    • 选择重要的任务

    • 逐渐增加任务难度

    • 让任务具有可重复性

    • 将任务分解，分步

  • 接受自己的犯错

    • 接受错误

    • 减少解释

第十章 药物疗法

  • 按需使用 ==> 表现型焦虑

    • 肾上腺素受体阻滞药物

    • 苯二氮卓类

  • 按时使用 ==> 广泛型社交焦虑

    • 抗抑郁药物

    • 苯二氮卓类

  • 草药疗法 ==> 降低焦虑

    • 圣约翰草

    • 卡瓦胡椒

# 第十二章 提升人际交往能力

  • 目光接触

  • 微笑

  • 倾听

  • 接近他人，主动互动

  • 闲聊的秘诀 ——> 说些无关紧要的话

  • 在冲突中合作

    • 不要通过逃避来处理冲突

    • 成为好的观察者

    • 让自己的行为影响他人

    • 保持冷静（避免愤怒/恐惧）

    • 不要采取暴力的手段应对冲突

# 第十三章 扩大朋友圈

  • 试着信任别人

  • 放低朋友标准

  • 联络旧友

  • 建立新交

# 第十四章 为自己的社交生活添加情趣

  • 停止拒绝邀请

  • 邀请别人到家里做客

  • 每周都有计划

  • 通过个人兴趣形成社交圈，建立联系

  • 和邻居多接触

  • 和家庭成员多联系

  • 用目标和活动增添动力

# 第十五章 为爱情引航

  • 留出差异的空间和时间

  • 先学会建立起好的友谊

  • 多尝试，多接触

  • 约会学习

    • 与约会对象相似的人练习对话

    • 多参与自己爱好的活动，增加机缘

    • 多参加活动和社交，培养经验

    • 和不同的人约会

    • 学习沟通和约会技巧的书籍
