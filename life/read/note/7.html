# 《如何阅读一本书》

《如何阅读一本书》 (opens new window)里面的方式方法对非资深阅读者有一定作用，下面是我对内容结构的概括。

  1. 基础阅读—能够阅读

  2. 检视阅读—知其大略

    • 用较快较浅的方式通读全书

    • 用各种技巧和碎片信息了解全书

  3. 分析阅读—系统阅读

    • 第一阶段——书说了什么

      • 分类

      • 精炼概括全书

      • 列出书的架构

      • 确定作者想解决的问题

    • 第二阶段——作者想说什么

      • 在关键字词的认识上与作者达到一致

      • 从重要的句子中抓住作者的主旨

      • 用书中相关的句子架构出作者的论述

      • 找出作者对问题的解答

      • 确定作者解决了和未解决的问题

    • 第三阶段——评书

      • 读懂再评

      • 不争强好胜，避免偏离真理

      • 注意作者得出的知识与观点的区别（相信知识终会接近真理，观点争论尽管不同却应该相信终能随真理的出现而解决）

      • 学会去发现作者的问题（知识不足，知识错误，不合逻辑和论证不完美）

      • 学会去查找资料，佐证你发现的错误

  4. 主题阅读—解决问题

    • 根据主题需要，择选相关书籍

    • 并列式读法，围绕主题进行阅读

    • 能通过阅读，解决该主题下的诸多问题，有所收获
