# 《心理学与生活》

菲利普·津巴多的《心理学与生活》 (opens new window)非常经典，内容也非常扎实，当时只对我感兴趣的几个章节做了笔记，分别是 1，2，15，16，17，18。

# 第一章 生活中的心理学

# 心理学为何独具特色

定义：关于个体行为和心理过程的科学的研究

研究目的：描述，解释，预测和控制行为以及提高人类生活质量

# 现代心理学的发展

# 心理学的历史根基

  1. 1879 年，威廉·冯特建立第一个正式实验心理学实验室，随着许多研究者的深入，围绕实验组织起来的心理学领域出现了结构主义和机能主义之争

  2. 早期心理学家证明了心理过程是由决定论原则支配的

    • 结构主义：强调由作为元素的的感觉构成的心理和行为结构

    • 机能主义：关注行为的机能，强调行为的目的

# 当代心理学的观点

  • 生物学观点：研究行为与脑机制之间的关系

  • 心理动力学观点：把行为看作是由意识和无意识动机驱动的

  • 行为主义观点：行为是由外部刺激条件决定的

  • 人本主义观点：强调个体内在的实现个人成长的能力

  • 认知观点：强调影响行为反应的心理过程

  • 进化论观点：把行为看作为在环境中生存而进化出的适应性

  • 文化观点：在文化背景中探索行为及其含义

# 第二章 心理学的研究方法

# 发现的背景

  1. 发现的背景指研究者在研究阶段，运用观察，信念，信息和一般的知识形成新的想法和理论。

  2. 心理学的理论力图理解大脑，心理，行为和环境之间的决定论关系

# 验证的背景：客观性保障

验证的背景指把证据拿来验证假设的研究阶段

# 存在问题

观察者偏见：观察者存在个人动机和预期（影响实验）

# 解决办法

  • 标准化：在数据的收集阶段使用统一的，一致的程序

  • 操作性定义：以测量它或决定它存在的特定的操作或程序来界定一个概念，在一个实验内使含义标准化

# 实验法

操作一个自变量来观察其在因变量上产生的效果

  • 存在问题

    • 混淆变量：实验中无意引入的因素影响到个体行为和对数据的解释

    • 期望效应：个体由于被暗示，被引导出了期望的实验反应

    • 安慰剂效应：个体对无效药物的乐观信念造成的健康增强

  • 解决办法

    • 双盲控制：让实验者和被试在实验中都存在一些未知因素

    • 安慰剂控制：让个体对无效的实验物充满信心以促成实验效果

    • 被试间设计：个体被随机分配到实验条件和控制条件下，来接受不同实验

    • 被试内设计：个体充当自己的控制组，然后对比实验前后效果（A-B-A）

  • 相关法

    • 相关法用于测试两变量的相关程度

    • 局限性在于不能得出因果关系

# 心理测量

# 维度

  • 信度

  • 效度

# 方法

  • 自我报告法：顾名思义

  • 行为测量法：研究外显行为和可观察，可记录的反应的方法

# 人类和动物研究中道德问题

尊重人和动物被试的基本权利

# 注意的问题

获得赞同，风险和收益的评估，对必要欺骗的规范化，事后解说

# 第十五章 心理障碍

# 焦虑障碍

属于心理障碍

# 类型

广泛性焦虑症，惊恐障碍，恐怖症，强迫症，创伤后应激障碍

# 原因

  • 生物学角度：人类机能本身对刺激有预备反应

  • 心理动力学角度：因为潜在的精神冲突和恐惧，这些症状是试图保护个体不受心里伤害，由无意识冲突爆发的意识中去的后果

  • 行为主义角度：某些行为会降低焦虑感，进而强化了这种行为

  • 认知角度：对引起焦虑事物的错误认知

# 心境障碍

属于情绪障碍

# 类型

抑郁症，双向障碍

# 原因

  • 生物学角度：脑状态的区别，与基因和遗传相关

  • 心理动力学角度：无意识冲突和童年早期形成的敌意情绪在抑郁情绪的形成中起了关键作用

  • 行为主义角度：经历的正强化和惩罚数量会导致抑郁，而悲伤情绪的被注意和他人的同情会对它起到强化作用

  • 认知角度：负性认知定势（消极的认知和行为预期），解释风格（习得性无助的状态）

# 性别差异

  1. 女性经历悲伤的时候回想到可能的原因及对她们感受的意义，男性则会集中注意于其他事情来分散自己的抑郁情绪。

  2. 这个模型提出，女性反应风格偏向于回味和思想，倾向于过度集中在自己的问题上，这增加了女性对抑郁情绪的易感性

# 人格障碍

类型：偏执型，表演型，自恋型，反社会型

危害：影响个人正常的社会功能

# 分离性障碍

类型：人格分裂

原因：可能与早年的虐待有关

# 精神分裂症

类型：紊乱型，紧张型，偏执型，未定型，残留型

原因：基因，能够遗传，风险基因与环境应激的结果

影响：生活质感的降低

精神疾病的烙印：人际排斥和社会拒绝

# 第十六章 心理治疗

# 治疗目标

诊断 ——> 确定病因 ——> 干预评估 ——> 确诊治疗

# 治疗师

  • 心理咨询师：心理辅导

  • 临床心理师：心理治疗

  • 精神病医师：心理疾病的医药治疗师

  • 临床社会工作者：进行社会活动的心理学家

  • 精神分析师：精神分析学的心理治疗师

  • 宗教顾问：宗教团体中负责心理障碍治疗的成员

# 治疗手段

# 精神动力学疗法

  • 传统学派的方法

    • 自由联想

    • 阻抗分析（回避和不愿言说的）

    • 梦的解析

    • 移情和反移情分析

  • 新弗洛伊德治疗

    • 考虑人对社会关系的需求

    • 客体关系理论（对引发心理问题的客体干预）

    • 自我概念的强调

# 行为疗法

  • 反条件作用

    • 系统脱敏法

    • 内爆疗法

    • 满灌疗法

    • 厌恶疗法

  • 意外事件管理——以矫正行为结果来改变行为

    • 正强化策略：对积极行为进行强化（奖励和表扬），以改变行为

    • 消退策略：对负面行为进行忽视和冷淡，以弱化行为

# 社会学习疗法

  • 榜样学习

    • 参照性尝试和改变

  • 社交训练

    • 具体情境下，我们应有的行为

# 认知疗法

  • 认知行为矫正法

    • 通过促成积极认知，改善行为模式

  • 抑郁治疗法

    • 消除消极的自动式思想，改变个人的思维加工和归因方式

  • 理性情绪疗法

    • 通过让个体克服非理性的狭隘观念，提高其自我价值感，摆脱不良信念体系的影响而达到自我实现

# 存在主义和人本主义疗法

  • 来访者中心疗法

    • 提供非指导性环境，让个人克服错误的学习模式，达到自我提高和自我实现

  • 格式塔疗法

    • 聚焦于个体整体性，旨在让个人在身心统一中表达自己的感受并认识内心的冲突

  • 婚姻和家庭疗法

    • 通过调整关系系统中的问题来治疗个体的问题，治疗的焦点在于改变夫妻间或家庭中的关系系统

  • 团体疗法

    • 团体小组治疗允许人们观察并投入到社会互动中，能够获得社会支持和安慰，降低了心理痛苦，也降低了参与和改变的困难

# 生物医学疗法

  • 精神外科手术

    • 对大脑施行外科手术

    • 存在副作用和不可逆后果

  • 电休克疗法

    • 对大脑进行电击

    • 存在暂时性副作用

  • 药物治疗

    • 包括抗精神病药物，抗焦虑药物和抗抑郁药物等

# 第十七章 社会过程与关系

# 情境的力量

人类的思维和行动受情境因素的影响

# 角色与规则

  1. 被安排扮演的一个社会角色，即使是在认为场景中，也能使个体作出一些有违他们信念，价值观和倾向性的行为

  2. 斯坦福监狱实验说明，情境力量会战胜习得的行为规范

# 社会规范

社会规范塑造群体成员的态度和行为

  • 从众

    • 信息源影响——通过观察学习，以作出准确行为

    • 规范性影响——使自己的言行认知与众人保持一致，就算被理智判断为错误

    • 迎合性影响——为维持自己认可的角色而去附和一些非主观的言行和态度

少数派很可能通过信息性影响对多数派意见产生影响

# 构造社会现实

  • 构造社会现实——指个体会依据自身的知识和经验解释情境

  • 社会知觉——受信念和期望的影响

  • 归因理论——指社会知觉者利用信息去生成因果解释的一种总体方式

# 归因理论的 4 种偏差

  • 基本归因错误——人在分析某些行为和后果时，高估了自己内心的认知倾向，低估了情境中的客观原因

  • 自利性偏差——对某些结果的成功归结于自己，失败归结于别人

  • 期望和自我实现预言——预期的评价会让个体产生变化

  • 行为确认——由假设信息造成的对目标行为和情况的预期和验证

# 态度，态度改变与行动

# 态度和行为

  1. 态度是对客体，事件和观念的积极或消极评价

  2. 态度只能在一定程度上预测行为

  3. 态度基础和论据类型是否匹配也影响论据的有效性

  4. 当态度容易获得或者比较具体的时候，态度和行为之间有较强的纽带关系

# 说服过程

  • 精细可能性模型——描述人们有多大可能将他们的认知过程集中在精心考虑说服性讯息

  • 中心路径——人们仔细思考说服性的沟通意见，因此态度改变与否取决于论点强弱

  • 外周路径——人们不怎么集中精力关注讯息，而是对情境中的表面线索作出反应

# 说服自我

人们往往会因自身的行为而改变态度

  • 失调理论——人们对自身言行不一的矛盾进行调和与辩解

  • 自我觉知理论——根据自身目前应该做什么和过去做的什么来产生自己的态度

# 依从

实现手段——互惠，承诺，稀缺，示范

# 社会关系

成人依恋风格影响关系质量

一个人对爱的关系的认同程度与密切和依赖程度有关

  • 喜欢

    • 产生原因——身体魅力，相似性，互惠

  • 爱

    • 相关因素——情欲，亲密，承诺

# 第十八章 社会心理学、社会和文化

# 利他行为

# 进化论原因

  • 利亲属：基因相近

  • 利他人：对互惠性存在与生俱来的期望；某些个体符合基因匹配对象

（个人总结——利他行为源于人类基因自私的本质）

# 亲社会行为

# 动机

  • 利己中心——行为有回报

  • 利他主义——基因相近或匹配和互惠性期望

  • 集体主义——集体利益

  • 规则主义——道德，宗教或习俗规则

某些情境下，不同的动机可能会发生冲突

# 亲社会的情境效应

旁观者的存在会分散情境中的责任感（例子：见死不救）

  • 原因

    • 责任被分散，没意识到紧急。没确定为紧急没感觉到责任

    • （道德原因）责任顾虑

# 攻击

# 原因

进化而来的保卫机制

  • 个体差异

    • 基因，荷尔蒙和人格等

  • 情境影响

    • 温度，被挑衅，挫折程度

  • 文化

    • 个体结构（西方独立性自我结构，东方依赖型自我结构，前者更具进攻性）

    • 文化中榜样信息（暴力）

# 偏见

# 根源

对社会的心理分类倾向——内群体和外群体

  • 人通常对属于自己的内群体持乐观肯定的态度，对外群体存在偏见和刻板印象

  • 即使是很小的线索，也足够形成偏见

# 刻板印象的影响

  • 造成某些偏见

  • 压制个体能力发挥

# 克服偏见的方法

  • 追求共同目标，形成合作

  • 友谊

# 冲突与和平心理学

# 服从权威的原因

  1. 情境在很大程度上控制个体行为

  2. 在情境中，人们服从权威的原因在于

    • 规范性影响

    • 信息源影响

  3. 具体有如下四点：

    • 人们倾向于做别人做或别人要求自己做的事，从而使自己被社会接受和承认

    • 人们容易接受暗示

    • 对如何不服从很迷惑

    • 实验情境下对权威的顺从实际上是人们固有习惯的一部分，是从小习得的

# 种族灭绝和战争的心理学

  1. 困难条件下，人们对于内外群体的区分会更固执，人们外群体更容易产生敌意

  2. 领导者习惯将敌人描述得没有人性

  3. 群体容易受到领导者煽动

  4. 战争中人们由于现实和不现实的原因，容易相信自己处于危险之中，因为害怕自己的家庭和国家受到威胁而选择为国家现身，加入战争

  5. 对各种政府及领导形式的研究表明，民主力量的扩大有利于整个世界

  6. 一些促进传统敌对国家相互交流的计划可以促进和平进程

  7. 那些旨在促进敌对间的交流，解决矛盾的科学项目正在引导人们迈向和平
