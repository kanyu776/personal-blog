# 《亲密关系》

《亲密关系》 (opens new window)只有前面4章做了笔记。

# 第一章 人际关系的构成

# 人类社会属性的核心

对亲密关系的需要

# 关系的基本构成要素

  • 文化经历

  • 人格

  • 遗传

  • 人际交往

# 亲密关系的性质

包含的6个特征

  • 了解

  • 关系

  • 相互依赖性

  • 相互一致性

  • 信任

  • 承诺

# 亲密关系中的归属需要

# 亲密关系的价值

  1. 与他人建立亲密关系是人类一种普遍而又强烈的内驱力

  2. 如果亲密关系的这种归属需要得不到满足，就会产生各种问题

  3. 具有亲密关系的人较之单身生活的人更幸福，健康长寿

  4. 亲密关系的质量会影响人的身心健康

  5. 我们的幸福感取决于归属需要的满足程度

# 人类为什么需要亲密关系

归属需要是人类长期演化的产物，逐渐成为所有人类共同的自然倾向

# 亲密关系变化的根源

# 社会经济发展水平

  • 支撑人的独立生活

  • 观念解放

# 新科技的影响

  • 生殖技术

  • 互联网

# 性别比率下的男女相对数量

  • 性别比率高，男多女少；反之，女多男少

  • 性别比率变化与人际关系规范有一定相关性

# 影响和决定人际关系的因素

# 文化的影响

  1. 它是规范亲密关系的社会标准

  2. 它是人们建立人际关系的基石，影响着人们对人际关系的期望，限定了正常的人际关系模式

  3. 同居，并不能保证随后的婚姻幸福。相反，增加了离婚的危险

  4. 有研究显示，同居人群的结婚率在下降，分手率却不降

# 个人经历的影响

  • 依恋类型

    • 评估维度：回避亲密和忧虑被弃

    • 类型：安全型，痴迷型，恐惧型，疏离型

    • 依恋的养成，取决于我们与照护者的交往经历和质量

    • 依恋类型虽然稳定持久，但还是可以被后天的交往经验改变，后者起更大的作用

    • 视乎人际交往经验的不同，我们习得的依恋类型既可以随时间发生变化，也可以永久保持稳定

    • 我们对亲密关系性质和价值的看法，取决于我们体验到的亲密关系经验

  • 天性气质

    • 个性和情绪性差异，影响着婴儿的受养护水平

    • 养护水平，又影响着婴儿依恋类型的塑造

# 个体差异的影响

  • 性别差异（生物性差异）

    • 性别认识存在刻板印象，性别差异被夸大了

    • 两性差异存在，但实际非常小

    • 两性性别内的差异通常大于两性之间的平均差异

    • 两性之间的相似性远大于差异性

  • 性认同差异（社会性差异）

    • 对性别角色能力适应性上的刻板印象

    • 对性别角色行为和特质的社会文化期望

  • 人格

    • 大五人格特质

      • 开放性，外倾性，尽责性，宜人性，神经质

      • 人格对人际关系的影响远大于人际关系对人格的影响

  • 自尊

    • 对自我的评价构成了自尊

    • 自尊是满足我们归属与需要演化而来的机制

    • 人际拒绝会损害人们的自尊，其他沮丧事件则不会

    • 我们的自我评价取决于人际接纳，关注和欣赏，而它又构成了自尊

    • 我们对自己的认识来自于与他人的人际关系，并影响着人际关系的后续发展

  • 人类本性的影响

    • 以演化论为基础，有三个基本假设

      • 性选择使人类成为今天的样子

      • 两性差异取决于过去面临的繁殖困境

      • 文化决定了演化的行为模式的适应性

    • 某些规律和性别差异之所以存在，是因为它们是心理机制长期演化的结果

# 第二章 研究方法

# 问题的提出

  • 描述问题和性质

  • 确定事件的因果关系和影响

# 参与者的选取

  • 方便性取样——同意者即可

  • 代表性取样——确保参与者符合研究整体

# 研究的设计

  • 相关设计——只能确定相关性，不能确定因果性

  • 实验设计——创造并控制参与条件

# 发展性研究设计

  • 横断设计——比较处在不同人生阶段或不同年龄人的发展阶段

  • 纵向设计——统一参与者年龄，长时间跟踪

  • 回溯设计——追踪事件的时间变化，询问参与者过去的经历

# 场景的选择

  • 实验室——可控

  • 自然环境——不可控

  • 克服研究困难的方法——角色扮演

# 数据的性质

# 数据应用性

  • 信度

  • 效度

# 数据获取方法

  • 自我报告

    • 形式

      • 书面问卷

      • 口头访谈

      • 非结构性日记

    • 问题

      • 参与者对问题的解释

      • 回忆或直觉困难

      • 参与者报告的偏差——自我服务偏差，社会赞许偏差

  • 观察法

    • 体验式取样——断式的，短时间观察长时的行为

  • 生理测量

  • 档案材料

# 结果的解释和整合

  1. 许多人际关系研究要对所获得的数据进行统计分析，以确定结果是否具有统计学上的显著性

  2. 如果结果偶发的可能性很低，那么结果就是显著的

  3. 数据的复杂性和问题

    • 配对的相互影响

    • 不同的分析水平

    • 影响人际关系的配对组合性

# 第三章 吸引力

# 吸引力基础：一种奖赏

# 奖赏类型

  • 与他人交往产生的直接奖赏

  • 仅与他人有关的间接奖赏

# 吸引力基础

与他人发生的有奖赏意义的经历

# 吸引力原因

不光与个人特征有关，还取决于我们自己的需要，偏好，愿望以及所处的情境

# 吸引力要素

# 长相——喜欢可爱的人

  1. 平均化的面孔几乎比组成它的所有但也个面孔更具吸引力

  2. 世界各地的人对于美丑的判断标准具有相当的一致性

  3. 已确定爱情关系的伴侣，他们的长相吸引力往往是类似的

  4. 亲密关系越严肃，投入程度越多时，匹配就越明显

  5. 与长相胜于自己的人约会的恋爱关系稳定性欠佳

# 临近——喜欢身边的人

  1. 当人们面对面交往时，在身体和心理上都更接近，这样的人际关系奖赏价值更高

  2. 要给临近的作用下一个定论的话，就是它能增强我们对他人的情感

  3. 方便

    • 我们很容易得到他们提供的各种奖赏

  4. 熟识

    • 曝光效应——重复接触他人能增加我们对他们的喜欢程度

    • 熟识能增加吸引力，但是过度接触则未必

    • 令人厌烦的人的重复接触，则只会雪上加霜

# 互惠——喜欢那些也喜欢我们的人

  1. 大多数人会用下面的公式来衡量自己对他人的实际兴趣，以及接近伴侣和建立亲密关系的可能性

    • 对未来伴侣的期望值 = 伴侣的外表吸引力 x 伴侣接纳自己的可能性

  2. 喜欢那些喜欢我们的人这一倾向是与吸引力的奖赏模型一致的

# 相似性——喜欢与我们相像的人

  • 相像种类

    • 首先是年龄，性别，种族，教育程度，宗教信仰和社会地位

    • 其次是态度和价值观

    • 最后是性格

  • 相异并不相吸——总体来说是否定的，“相异吸引”可以理解为相像起作用方式的一种巧妙体现，它可能误导了人们的认知

    • 匹配是广泛的过程

      • 如果相异看上去相互吸引，也许人们是在进行优势交换，价值适配

      • 两性的价值交换这种公平交易是演化心理学的核心观点——女性保障养育，男性保障资源

    • 发现差异需要时间

      • 很可能我们最初的相似的知觉是基于幻想和错觉，它需要时间检验

      • 知觉到的相像程度和关系满意度之间的相关，高于真实的相像和关系满意度之间的相关

    • 你就是我想成为的那个人

      • 那些我们渴望成为的人对我们拥有吸引力

    • 差异可能随时间减少

      • 习性磨合，造成了相像

    • 相异还能吸引——互补性

      • 要建立有效率的，称心如意的互补性需要时间

      • 需要牢记的重点是，相像的伴侣可能比不相像的人更能满足我们的需求

    • 相像具有吸引力的原因

      • 遇见和我们相像的人通常令人心安，具有奖赏价值

      • 相异看似相吸的原因或许不少，但事实上我们的确是物以类聚，人以群分的（相像具有奖赏意义，而相异则没有）

# 障碍——得不到的就喜欢

  1. 人们努力克服障碍以实现期望的基本倾向构成了一种“吸引力”

  2. 心理抗拒理论认为，如果人们失去行动或者选择的自由，会奋力争取重获自由。因而，如果我们面临失去某种事物的危险时，我们可能反而想得到更多

# 男女两性期望的理想伴侣

  • 在世界各地，人们评价未来伴侣有三条基本标准

    • 热情和忠诚：值得信赖，亲切友善，给予支持，善解人意

    • 吸引力和活力：长相俊美，性感，外向

    • 社会地位和资源：经济宽裕，生活安心

  • 两性在短期情人和长期伴侣之间的差异

    • 对短期情人，男性更看重女人的吸引力和活力

    • 对短期情人，女性更容易被男子气概，能引起性激动的男人吸引

    • 对长期伴侣，男性更看重友善，漂亮和善解人意的女性

    • 对长期伴侣，女性更看重品格，热情，忠诚，社会地位和经济资源标准，而不是长相和活力标准

    • 总体而言，男性专注于长相，女性专注于资源

# 第四章 社会认知

# 定义

一般指我们理解社会现象的知觉和判断过程，它决定了人们的感受和行动

# 第一印象

  1. 把人们的注意力导向某类新信息，进行验证性偏差的理解

  2. 影响对后来所获新事实的解释，导致人们不能客观公平，不偏不倚地处理来自他人的信息

  3. 第一印象与刻板印象塑造出对他人的整体印象

# 知觉的力量

人们很少会意识到，自身持有的信息和观点往往都是通过选择采纳的

# 知觉理想化

积极错觉——对伴侣

# 归因过程

  • 行动/观察者效应——对他人进行内部归因，对自身进行外部归因

  • 自我服务偏差——成功归功于自己，极力推脱失败罪责

  • 伴侣双方总的归因模式，能决定亲密关系的满意程度

# 关系信念

  • 宿命信念——对亲密关系本质属性的错误认识具有负面作用

  • 成长信念——更积极安全，对于两性关系更具适应性意义

# 期望

自我实现预言——我们对目标对象的期望和特定交互会引导其做出符合期待的行为

# 自我觉知

  • 自我提升——他人对自身能力，价值和吸引力的反馈

  • 自我证实——他人对自我概念的支持性反馈

  • 对于两性关系而言，人们更愿意选择能支持他们现存自我概念的伴侣

# 印象管理

对外在于他人的形象印象的维持

# 意义

  1. 我们在他人面前做的所有事情，其调控策略几乎都是围绕着印象管理的

  2. 对人们的社会生活有着广泛的影响

# 策略

  • 逢迎讨好

  • 自我推销

  • 恐吓

  • 恳求

# 亲密关系中

  • 人们管理印象的动机存在个体差异，这种差异具有重要的影响

    • 自我监控能力低的人，任何情境下的自我表现都很固定和一致

    • 自我监控能力高的人，任何情境下都能灵活的调整行为以适应关系环境

  • 在恋人和陌生人之间，在维持印象上，我们在后者更多

# 对伴侣的了解程度

  • 了解——我们并不如我们认为的那样了解自己的伴侣

  • 动机——关系初期对伴侣的认知更具洞察和准确，关系长期后似乎更不准确

  • 伴侣的易理解性——跟伴侣自身的性格和情绪外显性有关

  • 危险的知觉——人们会习惯于逃避和歪曲对关系质量的忧虑和不安

  • 知觉者的能力——情绪智力存在个体差异

  • 知觉者的影响——伴侣间带着相互的期望进行交互和磨合，会在一定程度上改变伴侣，符合期望和理解
