# 《男人来自火星，女人来自金星 2》

《男人来自火星，女人来自金星 2》 (opens new window)对两性差异和恋爱的理解很到位，三观非常正，并非技巧和套路系，是一本认真讲述两性差异和关系的书，颠覆了我以前的一些情感认知和态度，我觉得男女都应该看看这本书，下面是我的笔记和感想。

# 核心要点

  1. 恋爱的 5 个阶段：吸引，不确定，排他性，亲密性，订婚

  2. 女性吸引力特质：自信，善于接纳，积极响应

  3. 男性吸引力特质：自信，目标明确，责任感

  4. 情感反应的 4 个要素：不同的兴趣爱好，互补的需求，成熟，共鸣

  5. 男人适合追求，女人适合响应

  6. 男人通过付出得到满足感，女人通过接受来获得幸福感和满足感

  7. 男人应该克制展示倾向，学会倾听女人

  8. 女人应该勇于表达需求，积极参与到男人的事上（活动和兴趣）

  9. 学会克制，延迟满足

  10. 付出不应该超过给予

  11. 学会辨识不健康的情感反应，不过分依赖感觉

# 差异说

  1. 和谐的价值，并不意味着两个人拥有共同的兴趣爱好，甚至是相同的需求

  2. 差异并非障碍，它们恰恰是促进双方圆满的东西，它们让彼此拥有发展的空间，它应该成为一种激励和成长的源泉，而非障碍

  3. 差异存在的问题，只在于沟通的技巧

  4. 之所以会被不同的人吸引，在于对方满足了我们灵魂深处的渴求——我们的灵魂想要扩展，包容，超越自我。

# 个人理解与收获

  1. 差异说的界限——差异磨合与相互增进的前提在于，个体的包容性，两个人的相融性，存在这样的支撑，夫妻才有可能越走越像。包容性，即是对差异的容忍程度与偏向；相融性，即是相似的内在三观。

  2. 差异说的启发——应该尝试突破旧有的生活壁垒和自我模式，勇于去尝试一些新的东西，给自己带来一些不一样的改变，拓展和丰富

  3. 对不健康情感反应的理解——幼年的关系模板，在成年关系中的强迫性重复。因为与特定个体的关系/情感交互符合过往的关系模式，这也就是为什么有些人会重复喜欢上一些类似的人，脱离不了这个漩涡
