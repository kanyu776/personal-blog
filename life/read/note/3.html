# 《哲学是什么》

摘自罗伯特·保罗·沃尔夫的 《哲学是什么》 (opens new window)，主要是我个人认为笔记关键性的论述，只有第一章未做笔记

# 第二章 认识论

# 定义

认识论就是研究我们的认识方式，推理的法则，认识的界限以及我们用来判断一项假定的知识是不是真知的尺度和标准

# 笛卡尔的怀疑方法

  1. 笛卡尔的方法论，是一种探究和怀疑的方法，它要求质疑每一个不能被证明有绝对的确定性的信念

  2. 笛卡尔哲学带来的革命是，将传统形而上学对本体论和存在论的研究转向到认识论的研究，由关注知识客体，转向知识主体

  3. 笛卡尔以前的哲学都认为，形而上学优于认识论；笛卡尔的方法论，颠倒了研究的优先秩序，开启了新哲学发展

  4. 笛卡尔的我思论证产生了两个结果，唯我论和哲学研究的转向(由认识知识转向如何认识知识)

  5. 笛卡尔强调在获取知识的过程中，理性居于首位。我们的知识来自理智，理性或心灵，而不是感官

# 莱布尼茨与唯理论

  1. 真理和确定性的逻辑标准，逻辑真理——矛盾律，事实真理——“充足理由律”

  2. 事物的终极理由必定是一个必然性实体，而这个实体我们叫做上帝。

# 休谟与经验论

  1. 知觉分为那些直接来源于我们感官的与那些我们通过复制，重组，改变我们的诸印象形成的这两类

  2. 心灵包含有细小的，不可分的感觉“原子”，和这些感觉原子不可分的复制品，以及我们可以称作原子式感觉的“分子式”组合物

  3. 经验事物中的很多关联并没有真实和逻辑上的因果性

# 康德对唯理论和经验论的解决

笛卡尔的后继者们专注于用来判断心灵所思想东西的真理的标准，而且他们专注于心灵所思想的诸观念的来源，但是他们甚少关注这个中心问题，即正在思想的心灵是有意识的

  1. 认识基于意识，而意识则通过诸范畴统一了思想和知觉等意识内容

  2. 我们事实上从来没有关于实在的知识，而只有事物向我们显现的知识，并且心灵自身贡献的是形式，通过这个形式我们认识了现象——我们只能获得现象知识，无法获得实在知识

# 认识论的新转向

理性主义依赖理性作为知识的唯一来源，经验主义依赖经验作为知识的唯一来源

# 融贯论

  1. 信念不是孤立地得到其正当性的，而是通过它们的各个信念组成一个连贯的系统，在此系统中各个信念是相互一贯的而且相互依赖

  2. 一方面是各个信念，另一方面是与这些信念例如感觉不同的但却是其依赖的一个基础

# 回归实用主义

  1. 实用主义是一种以实践为导向的哲学

  2. 一个理念的真理的最终检验是它在实践中的有用性——一个断言的真理性产生于它在实践中的结果

  3. 厌恶浮夸的形而上学沉思，在认识论中反对把知识等同于绝对确定性的笛卡尔观点

  4. 知识是行动的向导，它对于我们有帮助作用，并且是令人满意地用来组织我们的经验材料的一种工具

# 认知科学的冲击

  1. 认知科学典型地依靠信息处理来作为他的模型从而思考智能活动

  2. 从认知科学的观点来研究认识论，它的贡献在根本上是有缺陷的，因为它已经预先假定人类的认知实际上就像一台信息处理机器

# 第三章 形而上学与心灵哲学

  1. 形而上学，是对存在事物的本性的哲学研究

  2. 霍布斯提出一个现代版的古代理论，即宇宙是由细微的不可见的物质粒子或原子组成的。通过他的物质原子理论，霍布斯寻求解释人类思想，感觉，选择和行动的途径

  3. 由心灵和身体之间的关系而提出的一个中心问题是自由意志。

  4. 霍布斯是一个决定论者，主张我们的所有行为都是由原子的运动所决定。

  5. 康德论证道：现象领域，即数学和科学的世界，是彻彻底底地被决定的。但是独立实在的领域是出于自然的因果关系链条之外的。他通过对现象与实在之间的区分，从而试图表明科学的决定论能够与道德所要求的自由相容

  6. 身心之间的关系有三种主要的理论：二元论，唯心主义和唯物主义。使用复杂巧妙哦逻辑分析，一些现代哲学家为传统的唯物主义辩护

  7. 精神和身体的同一性理论——每一种精神现象（状况，事件或过程）和某些身体现象是同一的，同时也是偶然性的

# 第四章 科学哲学

1.观察的搜集——科学观察必须是理论中立的，观察者接触经验时不要带上前设或偏见，记录下呈现在他感官面前的任何东西——培根主义

  1. 观察必须受到理论的指导，诉诸实验——观察与理论之间的关系

  2. 一个成功的科学理论的本质就在于，尽管它是可以被否证的，但是它现在还没有被否证。——波普尔

  3. 所有的观察都是有选择的——波普尔

  4. 常规科学，范式和科学革命——托马斯·库恩

  5. 科学革命就是“非累积性的发展阶段，在此阶段中，一个旧的范式被一个不相容的新范式整个地或部分的取代”

  6. 科学是一种“社会建制”是指，科学是一种活动或一种实践，这种活动或实践被组织进各种社会角色

  7. 作为一种社会建制的科学有其特定的行为规范或行为标准，这些规范或标准从老手传递给新手，并且被其专业成员正式地或非正式地加以强化

  8. 只有科学作为一种社会制度这个观念能够引领我们去正确评估个人的科学努力

  9. 从科学作为社会建制的角度看，一个科学事实取决于社会习俗。除非被科学共同体所接受，通过征引，复制实验，以及最后合并成著名的“事实”的标准来标明，就不存在其他的检验方法了

  10. 科学哲学的发展，经由培根以观察集合为开端，过渡到理论指导实践——实验，再到库恩对传统科学的渐进式迈进的否定，指出科学的“替代性”——新范式取代旧范式

  11. 最后，科学强调社会建制，使得关于科学价值的判断倾向于社会判断而不是逻辑判断

# 第五章 伦理学

# 康德于义务命令

  1. 把真正的道德问题看作是在面对诱惑和欢娱时永恒地奋力去做我们知道是正确的事情。

  2. 信仰和道德都必须服从批判的精神

# 思考伦理学的三项理由

  1. 追求行为中绝对确定的，普遍正确的第一原则，从而能够经受得住怀疑主义者和相对主义者的挑战。

  2. 追求一种推理的方法或推理的过程，来帮助我们决断这些难办的事情和其他真实世界中的道德选择

  3. 追求一种好的生活，这种生活包含德性和幸福的真正实现

# 伦理分歧与定言命令

  • 哲学家基本上有三条途径来处理道德规范差异

    • 否认差异的存在

    • 承认差异的存在

    • 承认差异的存在，但是坚持认为不管怎么样都有一些道德原则是真的

  • 科学的实验和对数据的检验是井然有序的，而道德论争则具有无序的，直觉的，非事实性的特征

  • 康德的定言命令

    • 要按照你同时也能愿意它成为一条普遍法则的那个准则来行动

    • 当理性向激情低头的时候，它就丧失了它对尊荣和尊严的要求

    • 如果理性能够它自己服从的法律，如果理性自己能够写下约束它自己的定言命令，那么在每一个服从行为中它将保留它自己的自由

    • 理性自律的原则是定言命令的另一种说法

  • 康德伦理学的三个核心理念

    • 人是理性的被造物，能够思考他们所面对的诸种选择，并且根据理性从中作出选择

    • 人拥有无限的价值和尊严，从而使得他们超越于世界上所有仅仅是有条件地有价值的事物之上，康德把人称为“目的自身”

    • 人作为理性的目的自身，是道德法则的立法者，因此他们对于义务的服从并不是一种奴隶般的顺从行为，而是一种有尊严的自律行为

  • 康德提出了一个道德原则叫作“定言命令”，它对于所有人在任何情况下都是绝对正确的原则。伦理相对主义者与伦理绝对主义者否认有这样普遍有效的原则存在

# 功利主义

  1. 通过计算比较，使得幸福得以最大化

  2. 为解决伦理困境提供了一个“道德原则”，即“最大多数人的最大幸福”原则

# 德性伦理学

倡导专注于对好的生活的探索，这种生活综合了德性和幸福，并使它们在人类中真正实现

# 女权主义对伦理学的批判

  • 贾格尔研究的几个特点——所有这些反对意见都指向现代自由传统的基本缺点

    • 现代道德哲学中自由主义传统的观点，是出自一个西方男性户主，上层或中层阶级，因此很可能是白人。从这个视角来看，它是讲究优劣，区别对待的。

    • 对普遍性，个特性和无私性的质疑，这些都是构成现代自由传统中的理论模型

    • 拒绝道德哲学家在他的研究中把自己当作唯一的沉思着模型

    • 认为道德辩护是获得“道德的观点”，一个理想观察者的观点，或一个阿基米德点

  • 西方伦理理论传统的女权主义批判家们争辩说，各种各样的主流理论全都体现了男性的或家长制的观点，而未能表达女人的经验与的洞见

# 第六章 社会与政治哲学

# 密尔与古典放任自由主义

  1. 所有的人基本上都是明智的，理性上自私的行为者——这就是说，我们以最广泛和最快乐的方式满足我们的各种欲望，而且我们使用我们可以得到的诸种资源——金钱，天赋，权力——以可能最有效率的途径来满足欲望

  2. 理性的自私行动存在两大障碍——迷信和无知

  3. 教育，是功利主义者针对这对孪生敌人迷信和无知所需要的武器

  4. 功利主义结合了个人动机的心理学理论，有关善的道德理论和启蒙的教育理论，从而产生了我们今天称之为自由民主的政治理论

  5. 功利主义体现的最后一个元素——自由放任主义经济理论

  6. 只要市场中经济的行动者以一种理性而自身的方式行动，那么我就能够计算他们的各种行动而不需要对他们的个人性格拥有任何在先的知识，也不需要搜集有关他们国王行为的详尽信息

  7. 功利主义者捍卫自由市场或自由放任原则，即消费者和生产者应该在市场中自由地交易，而没有政府的干预

  8. 密尔对正统改革学说论题的三种质疑方式

    • 否定边沁的平等主义主张（即边沁认为任何快乐，就其自身或处于自身而言，都与其他的快乐一样的好）——密尔强调快乐是有区分的，高尚和低级)

    • 强调在市场中实行有限的政府干预，并且在贸易和商业中消除所有管制

    • 关注政府干涉它的公民私生活的权利

# 社会主义者对资本主义的批判

  1. 社会主义者如卡尔·马克思批判资本主义是非理性的，剥削工人阶级，腐化了劳动过程，使得劳动成为痛苦，苦难而不是满足感的来源，并且把经济导向更大的商业危机。

  2. 马克思主张，通过把社会的生产资料所有权和控制权交道工人阶级手里，社会主义革命就能够终结资本主义

# 卢梭与社会契约论

人民与国家的关系是一种契约的关系，一种直接的关系，国家代表民意，个人与国家之间没有任何“中介”

# 多元主义国家理论

  1. 所有人都属于众多不同的团体，没有人实际上是作为一个人面对一个国家的。各种团体的人们他们总是作为团体来起作用

  2. 美国民主的多元主义理论考虑到了两次大选之间利益集团的政治活动。它主张，通过这样的政治活动，公民利益的复杂性和强大程度就得到了表达。

  3. 多元主义的民主理论并没有给所有人利益的公平表达腾出空间

# 对国家社会契约论的种族批判

  1. 社会契约论便利地忘记提及有色人种，社会契约的参与者都是用强迫的控制加在他们身上

  2. 古典的社会契约理论及其现代变体，完全忽视了白人对于非白人的宰割和剥削这是过去四个世纪以来世界历史上的最重要事实。

  3. 一个替代性的国家模型，被查尔斯·米尔成为“种族契约”，他试图捕捉并理解这一事实
