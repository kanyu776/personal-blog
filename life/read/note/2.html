# 《社会心理学》

来自阅读《社会心理学》 (opens new window)后的笔记纲要，缺记第16章。

# 第一章 社会心理学导论

# 定义

探讨人们的思想，情感以及行为如何因他人真实或想象的存在而受到影响的科学研究（人如何被社会影响的学科）

# 主要关心

人们如何受到他们自身对社会环境的诠释或解读，而不是客观社会环境

# 学科关系

社会学——社会心理学——人格心理学

  • 社会学，从宏观角度解释问题

  • 社会心理学，从社会互动的心理角度解释问题

  • 人格心理学，从个体差异层面思考问题

在生活中，面对现象我们惯于从人格角度解释和思考问题，忽略了情境，犯了基本归因错误

基本归因错误——高估内在性情对行为的影响，而低估情境因素的思考倾向

# 社会情境的主观性

  1. 人如何解读情境（比如：对异性微笑的理解是礼貌，还是好感）

  2. 完形心理学想告诉我们，应该着眼于知觉的现象学即事物呈现在人心里的样子，而而不是基本的动机

# 情境解读的两个基本动机

# 自尊取向

  1. 维持良好自我感觉的需要——感性取向

  2. 为了完成自尊取向，我们会倾向于扭曲事实

# 社会认知取向

对准确性的需求——理性取向

# 其他动机

  • 对控制的需要

  • 对奖励的需要

  • 生物性驱动

# 自证预言——阻碍了正确知觉的心理学原理

人会依据既定的认知去对待人事，导致人事朝这种认知去发展，最终实现“预言”

# 第二章 方法论

# 观察法

描述社会行为——基于观察

# 相关法

预测社会行为——基于相关性

# 实验法

解释因果关系——基于因果联系

  • 内部效度——正确性

  • 外部效度——适用性

# 社会心理学研究的新思路

# 文化与社会心理学

  1. 重视个体的独立还是重视社会性的相互依赖，这反映了我们的文化价值观

  2. 民族志这种典型的人类学研究方法正成为社会心理学研究的一种方法

# 进化心理学

很多行为也许根植于自然选择后的基因结果

# 社会神经科学

行为与大脑活动的秘密

# 第三章 社会认知

# 定义

我们如何思考社会性世界——人如何看待自我与社会

# 自动化思维

  1. 基于图式和心理学策略

  2. 无意识思维

  3. 文化对图式这种心理认知结构的决定和培养

# 控制性思维

有意识，有目的，主动并努力的思维

# 改变人类思维

思维可以通过训练而被改进

# 第四章 社会知觉

# 定义

我们如何理解他人

# 非语言行为——非语言知觉

  1. 表达情绪的面部表情，是进化的结果

  2. 文化限制和规范了非语言表达

  3. 对非语言行为解码有时会出错

    • 情绪混合

    • 文化区别

# 内隐人格理论——图式联合的知觉

# 定义

以图式联合多种人格特质，对人格进行整体知觉和认识的理论——图式填补了认知上的空白

# 认知方法——归因理论

  • 内部归因——主体人格

  • 外部归因——情境解释

  • 共变模式——知觉者如何在内部归因和外部归因之间进行选择

# 认知偏见

# 一致性偏见

  • 人们相信行为和性格相一致

  • 原因：行为比情境有更高的知觉显著性

  • 个人主义和集体主义文化都存在一致性偏见，集体主义更易认识到情境解释

# 当事人与旁观者差异

  • 当事人进行外部归因，旁观者进行内部归因

  • 原因：人们在归因时使用了心理捷径，包括知觉显著性和信息便利性

# 自利归因

  • 将成功归因于内部，将失败归因于外部

  • 原因：维持自尊，在别人面前营造正面形象，对自己过去表现的了解

# 防卫归因

  • 使我们不受脆弱及死亡等感受困扰的解释

  • 善恶有报——公平世界的信念

# 文化与归因偏见

  • 有研究显示，在自利归因与防卫归因上具有跨文化差异

  • 东方集体主义文化倾向外部归因

  • 西方个人主义文化倾向内部归因

# 第五章 自我知觉：在社会情境中理解我们自己

# 自我概念

指自我的内容，即关于我们自己是谁的认识

# 自我觉知

指思考和理解我们自身，注意力关注在自身的结果

# 自我的4个功能

自我认知，自我控制，自我展示，自我满足

# 自我认知

# 方法

  • 内省

    • 会对认知有所遗漏

  • 观察自我行为

    • 存在过分理由效应，推断问题基于外部动机，降低了我们对内在动机的考虑和注意

    • 很多问题属于情绪唤起的错误归因

  • 通过他人

    • 包括和他人比较与采纳他人观点两种手段

# 自我定义存在文化差异

  • 西方人倾向于独立自我观

  • 东方人倾向于相互依赖的自我观

# 自我定义存在性别差异

  • 女子拥有更多关系上的相互依存性，更关注亲密关系

  • 而男子则具有更多集体上的相互依存性，更关注他们在较大团体中的成员资格

# 对自我能力的认知心态

  • 固定心态：自己的能力是固定不变的

  • 成长心态：自己的能力是不断积累变化的

# 自我控制

自我控制需要能量，在赋予一个任务其所需的能量后，另一个任务的执行能力就会相应减弱

# 自我展示

  • 印象管理：人们试图给别人展示出自己想要表现出的印象

  • 策略：逢迎和自我妨碍

不同的文化都试图展示最好的一面，只是对“好”的定义不同

# 第六章 合理化行为的需要：减少失调的代价和收益

# 自我满足

人有合理化自我的倾向（维持良好自我的形象）

# 认知失调理论

  • 定义：当人们的行为和认知遭遇不一致时产生的不适感

  • 减少认知失调的三种方法：

    1. 改变自身行为，以符合不同认知

    2. 改变不同认知，合理化自身行为

    3. 增加新认知，合理化自身行为

降低认知失调的过程是无意识的

# 面对认知失调

  1. 为了避免认知失调，我们通常容易先入为主的理解和加工信息，牺牲理性思考和行为

  2. 决策失调，选择的唯一性致使我们会通过强化选择结果，以降低失调，不论选择有多糟糕

  3. 合理化努力，我们会合理化自身的努力，以降低失调，尽管努力的东西可能并不好

# 非充分合理化

  • 内部合理化：改变自身，降低认知失调

  • 外部合理化：存在于个体外部，为不一致的个体行为寻找理由和解释

反态度辩护，扭转态度和认知，使之合理化

# 其他

  1. 较之外部合理化明显的情境，内部合理化对个体的影响更长久

  2. 社会实践

    • 可以制造言行不一来引发认知失调，使目标对象为降低失调而促成改变

  3. 善行与恶行

    • 通过让别人帮助你来让他们喜欢你（人们需要合理化他们帮你的事实）

    • 战争和屠杀（通过贬低受害者，以降低认知失调）

  4. 负面调节

    • 对错误认知进行合理化，甚至固化态度和观念

# 第七章 态度与态度改变：影响思维和情绪

# 定义

指一个人对人，事物和观点的持久性评价

# 来源

经验

# 态度

  • 以认知为基础的态度

  • 以情感为基础的态度（可通过经典条件作用和操作性条件作用形成）

  • 以行为为基础的态度

# 知觉分类

  • 外显态度——来源于近期经历

  • 内隐态度——来源于童年

# 态度改变——通常是社会影响的结果

# 通过改变行为来改变态度——重返认知失调理论

操作：通过让人们参与到反态度的行为辩解中，以致于缺乏外部合理化，固定态度与行为的一致性

# 说服性沟通

说服性沟通取决于：

  • 信息来源（沟通者）

  • 信息本身（内容）和听众

精确可能性模型——区分了人们何时会采取说服的中心路径，何时采取说服    的外周路径

中心路径——专注于沟通逻辑和论据（态度改变更持久）

外周路径——专注于表面特征

个体能力和与个体的相关性决定了路径选择

# 情绪与态度改变

引发恐惧的沟通会导致持久的态度改变

情绪启发式——以情绪作为自己对某事物的态度和线索

情绪改变态度的有效性取决于态度类型

  • 情绪诉求（社会认同）

  • 认知诉求（实用性）

# 对个人想法的信心与态度改变

只要你的论证有说服力，能让人们对你提供的信息产生信心

# 西方文化强调独立与个人主义，而亚洲文化则强调互倚与集体主义

强调个性与自我完美的广告更易在西方文化中发挥作用，而强调个人所属的社会团体的广告会在亚洲文化中产生效果

# 抗拒改变

# 方法

态度的预防免疫，警惕产品内置和拒绝同伴压力

# 抗拒理论——说服产生的反作用

当人们的自由选择受到威胁时，人们会体验到一种不快，以致抗拒

# 态度预测

# 预测自发行为

人们关于态度对象的经验越直接，人们自发行为与他们态度的一致性越高

# 预测有意行为——对特定行为的态度

  • 主观规范（对他人认识自我行为的理解）

  • 容易程度（把握）

# 第八章 从众：影响行为

# 定义

当人们由于真实或臆想的他人影响而改变自己的行为时，将产生从众行为

# 原因

  • 信息性社会影响——想知道“正确”情况的需要

    • 发生于情境模糊或权威存在的时候

  • 规范性社会影响——希望被接受的需要

    1. 当我们为了使自己和他人相称而改变自己的行为时，它就发生了

    2. 它通常导致个人的公开顺从，但不一定会导致个人的私下接纳

    3. 拒绝的可能性：当保持正确非常重要的时候，当少数人观点始终一致的时候

    4. 拒绝它的后果：被人嘲笑，排斥甚至被团体抛弃

    5. 它的日常影响：习惯，文化，日常和观念

    6. 发生的可能性：三人左右的团体，集体主义文化最容易发生

# 服从权威

# 原因

  • 信息性社会影响

  • 规范性社会影响

  • 个人责任感的丧失和情境的快速节奏

# 第九章 团体过程：社会团体的影响

# 团体的好处

  1. 能满足人类的一些基本需要

  2. 帮助我们确认自己的身份

  3. 帮助建立社会规范，即规定了可接受行为的内隐或外显的规则

# 团体与个人行为

# 社会促进

他人在场为我们添加活力

# 社会懈怠

他人在场使我们放松

# 影响过程

他人在场 -> 个人行为会被评估 -> 警觉，评价焦虑，分心冲突 -> 唤醒状态 -> 简单任务表现更佳，复杂任务表现更糟

他人在场 -> 个人行为不会被评估 -> 没有评价焦虑 -> 放松状态 -> 简单任务表现跟更糟，复杂认为表现更佳

# 去个体化

当人们处在群体而不会被觉察的时候，对行为的规范遵守就会放松

# 影响

个人责任感下降，促进个人对团体的服从

# 团体决策

# 过程损失

  • 团体互动抑制了良好的解决办法

  • 团体会集中于共有信息，忽略一些独特信息

# 团体极化

团体决策比个人决策更易极端化

# 团体领导

领导权变理论——领导的有效性取决于，领导者的风格是任务导向型，还是关系导向型

领导的有效性——是领导者类型与工作环境的性质共同决定的

# 第十章 人际吸引：从第一印象到亲密关系

# 产生吸引的一般原因

# 时空接近效应

# 相似性

相比互补性，相似性更有影响力

# 互惠式好感

一般来说，我们会喜欢那些表现得喜欢我们的人

# 人际吸引理论

# 社会交换理论

定义：人们对于人际关系的感受取决于他们对这段关系的收益与成本的知觉

基本概念：收益，成本，结果和比较水平

# 公平理论

双方收益与成本相当时，双方最快乐

# 爱的定义

激情之爱和伴侣之爱

西方认同于激情之爱，东方更认可伴侣之爱

# 进化与爱

  1. 进化心理学试图以根据自然选择原则而随着时间进化的基因因素来解释爱情

  2. 这个取向提出，两性为对方身上不同的特征所吸引是出于最大程度上保证繁殖成功的考虑

  3. 男性被女性外表吸引，女性被男性的资源所吸引

  4. 爱的进化论取向的基本观点是人类从祖先那里不断进化一些策略以帮助人类繁育健康的后代来保证物种延续

# 亲密关系的依恋类型

定义：基于个体在婴儿时期与主要的教养者之间的关系发展而来的个体对人际关系的预期方式

类型：安全型，回避型和焦虑/矛盾型

  • 我们在生命早期所建立的联结关系的形态会影响到我们成年时所建立的关系

  • 我们在婴儿和儿童时期习得的依恋类型会伴随我们一生，并且将会泛化到我们与其他所有人的关系之中

  • 具体的依恋类型，部分受到基因的影响

  • 依恋类型是灵活的，可以被改变的

# 长期人际关系中的社会交换

在长期关系中，忠诚依赖的指标

  • 满意度

    • 得到的收益

    • 付出的成本

    • 收益 - 成本 = 满意度

  • 投资水平

    • 对现有关系的投资程度

    • 时间，精力，金钱和感情

  • 替代性选择

    • 是否有替代性的关系选择

    • 替代性关系的预期收益是否大于现有关系的满意度

# 长期人际关系中的公平性

  • 亲密关系有着交换和共有的特征

  • 交换关系——受到对公平的关心的控制，主要关注自己能从帮助他人中得到什么

  • 共有关系——主要关心他人福利，较少关注自身的需要和收益

# 第十一章 亲社会行为：为什么人们助人

# 为什么人们助人

# 进化心理学：本能与基因

  • 亲属选择

  • 互惠规范

  • 继承的社会规范

  • 群体选择

# 社会交换：助人的成本与报酬

人们的助人行为是为了使报酬最大化和成本最小化

# 移情与利他主义：助人的纯粹动机

  1. 当人们对他人产生移情（与他人经历共同的事件和情绪）后，人们会处于出纯粹的动机去帮助他人

  2. 人们会受到利他主义的影响，产生帮助他人的愿望，即使需要付出代价

  3. 依恋类型：涉及人们对自己和他人关系使之的一种期待，而这由他们在婴儿期和教育者的关系模式所决定

  4. 当人们感觉到自己的依恋类型是安全型的时候，面对他人往往有较高的移情感受水平

  5. 移情现象的另外一个决定因素是我们最近是否受到自己所在乎的团体的接纳或排斥

# 为什么一些人比其他人更多助人

# 个体差异

利他人格

# 亲社会行为的性别差异

与男性喜欢表现出骑士风格和英雄主义不同，女性更喜欢在高承诺水平的长期关系中提供更多的助人行为

# 亲社会行为中的文化差异

所有文化中的人都更可能帮助他们认为是内团体成员的人，而较少可能帮助他们认为是外团体成员的人。

文化因素在决定人们划分内团体和外团体之间的界限清晰程度时起作用

# 宗教和亲社会行为

  • 那些宗教信仰者比非宗教信仰的人报告更多的主助人行为

  • 只有在可能向他人或自己显示虔诚和善良的情况下，宗教信仰者才比非宗教信仰者更倾向于帮助

  • 心境对亲社会行为的影响

    • 人们在体验好心情的时候更容易做出助人行为

    • 不过，这种情况在人们体验坏心情的时候也发生

# 什么时候人们会助人

# 环境：乡村与城市

密集型城市中的人们更少地做出助人行为

城市过载假设——住在城市中的人经常被刺激所轰炸，使得他们独善其身以避免被信息淹没

# 居民流动性

长期居住在某一社区的人比刚搬进该社区的人更愿意参与亲社会行为

# 旁观者效应

人在紧急事件中助人需要满足五个方面的条件：

  • 注意到事件

  • 解读为紧急事件

  • 承担个人责任

  • 知道如何帮助

  • 决定去帮助

# 第十二章 侵犯：我们为什么伤害他人

# 原因

  1. 一些人认为人类天生就是具有攻击性的。相反，另外一些人认为攻击性是可塑的，是由个体所处的社会环境决定的

  2. 进化心理学认为攻击性是内在的，因为其对于人类生存有着重要重要，它会增加人类自身再生产的机会；社会心理学家则认为，尽管攻击性行为对于人类来说是与生俱来的，但其发展和表现会受到情境和文化因素的影响和塑造

  3. 不同文化下的个体的侵略性不同。核心在于多种因素影响着文化是否促进攻击行为的发生

# 神经生化因素对侵犯的影响

# 性别差异

在受到刺激的时候，男性比女性更容易表现出攻击性。但是明确的挑衅性会弱化这种两性的攻击性差异。

# 酒精与侵犯

由于酒精会降低社会抑制，所以会增加供给性行为

# 疼痛和不适

当人们经历痛苦的时候，他们更容易表现出攻击性。高温，潮湿或者是异味等会造成躯体不适感的环境都可能增加个体的攻击性

# 社会情境与侵犯

  1. 挫折感，被激怒和报复，攻击性物体的提示作用，社会赞许，模仿和侵犯，影像媒介中的暴力展示，都是社会情境下的暴力性促因

  2. 接触暴力色情会增加对于针对女性性暴力的接纳程度

# 如何减少侵犯

  1. 只有当惩罚及时和确切的时候，才能起到很好地震慑犯罪的作用

  2. 通常情况下，对愤怒的宣泄弊大于利。

  3. 及时意识到自己的愤怒并且以一种建设性的方式来对待它会是一种比较有效的方法

# 第十三章 偏见：原因与消除

# 定义

一种对特定团体成员的敌对和负面态度，只有对方属于那个团体

  • 认知要素——刻板印象

  • 行为要素——歧视

# 起因

  • 思维方式——内团体和外团体的区分倾向

  • 归因偏差——高估个体的素质作用，容易以偏概全

  • 资源分配——资源分配导致的负面偏见

  • 从众行为——规范性从众，避免被团体孤立和排斥

# 解决办法

  • 接触——让内团体和外团体充分相处

  • 接触所必需的6个条件

    • 相互依赖

    • 共同目标

    • 同等地位

    • 非正式的人际接触

    • 频繁接触

    • 平等的社会规范

# 第十四章 实践中的社会心理学之一：获得可持续的未来

# 应用社会心理学研究

  • 利用实验方法

  • 社会心理救援

# 利用社会心理学实现可持续发展

  • 解决社会困境——公共困境(适当限制个人所用，保证公有资源充足)

    • 传递并改变社会规范

  • 追踪消费——让人们了解自己的消费

    • 引入小竞争

    • 诱导伪善

  • 消除实现过程中的小障碍——保证具体操作便利

# 幸福和可持续的生活方式

  1. 幸福一部分取决于天生的内在气质，一部分取决于外在因素：它包括人际关系的质量，自身追求和帮助他人

  2. 在人们满足了生活的基本需求之后，拥有更多的金钱几乎不能提升幸福感

# 第十五章 实践中的社会心理学之二：社会心理学与健康

# 压力与人类健康

当人们感到无法控制负性事件时，他们就感觉到了压力

# 定义

当人们感到无法应对环境的要求时产生的负性体验和负性信念。

# 知觉控制感

对自我的把控感和驾驭能力

# 自我效能

相信自己有能力采取特定行为以达成期望目标的信念

# 习得性无助

使人们对于负性事件做出稳定的，内在的，全面的归因的思维和心理倾向

# 压力的应对

  • 社会支持：获得他人的帮助

  • 敞开心扉：弄清创伤性事件的意义

知觉控制感强和自我效能高的人，对于压力有更好的免疫性

# 应对压力的性别差异

  • 男性在应对压力面前更倾向于表现出反击-逃避回应

  • 女性在面对压力时则会出现照顾-表现友好回应
