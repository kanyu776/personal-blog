# 阅读

这里是我对于阅读相关的内容进行的总结和整理，分为笔记，脑图和收获 3 个模块，下面是我的阅读推荐：

  1. 《精神分析引论》 (opens new window)

人的失常行为，梦境解析，意识与潜意识机制的经典理论诠释，能够加深人对精神病和心理问题的理解，会不由自主的被弗洛伊德带入深邃，智识提高。

2.《李天命作品集》 (opens new window)

一套非常潇洒的逻辑学作品，说潇洒是因为李天命的风格，体现于他的思考方法和人生哲学，让我意识到了网络中的情绪逻辑，日常言语中的颠三倒四，能够将你的思维和逻辑清晰化，准确化和严谨化，同时还能让你习得一种非常潇洒和智慧的人生观，受益匪浅，强烈推荐！

3.《四书五经》 (opens new window)

消除国学神秘感，领略古代儒家思想和风格，慎独修身，自我提高，增加国学文化常识，不可不看

4.《论人类不平等的起源和基础》 (opens new window)

短小精悍，语言和思想阐述的非常到位，人类不平等的起源的论述非常经典，读起来非常流畅易懂

6.《社会契约论》 (opens new window)

以现在的眼光，当下的民主思想和政治理论来看本书，好像稀松平常，只是在重复教科书中的理念，但是真正的根源是来自于此，全面系统铺开，值得一读

7.《六祖坛经》 (opens new window)

非常通俗的佛学讲道，能够让人初步体会到佛家思想的达观与智慧

8.《心经》 (opens new window)

‘空’和‘色’真不是世俗作想，而是代表了佛家的思想观和重心，一本精小的佛学经典，领略佛家面貌。

9.《道德情操论》 (opens new window)

亚当·斯密一部非常经典的伦理学书籍，给我印象最深刻的不是整套思想理论，而是其语言的细密和到位，总是能非常细致地讲明一些日常言之不出的道理，像是抽丝剥茧一般，让你佩服其能将事情讲到如此清楚的地步

10.《平凡的世界》 (opens new window)

语言文字非常平实，情节过渡非常到位，故事感人励志，贴近生活与心灵，让人振奋和顽强

11.《人生的智慧》 (opens new window)

文字简炼，思考深刻，常常能一语中的，直点要穴，让人醒目彻悟，改善认识，更好的过活

12.《直视骄阳·征服死亡恐惧》 (opens new window)

引用各家对于死亡的思考理论，自我表露和深刻剖析，深入浅出，让人眼界豁达，心智开朗

13.《尼采:在世纪的转折点上》 (opens new window)

只要稍微熟悉尼采的人，应该都会觉得这部分作品对尼采及其作品的解读不赖，对于理解尼采思想和尼采其人很重要的一本书

14.《少有人走的路 1》 (opens new window)

教你如何直面自我，真正了解爱是什么，理解成长过程和成年困境，明白心理疾病和部分神经官能症的原理原因，一部结合实例述说的心理学书籍，帮你开启心智成熟之旅

15.《好好爱自己》 (opens new window)

类似散文，接近语录式的文字作品，但是关于情感，婚姻和爱情的纠葛和困惑，语言非常精髓到位，一针见血，女同胞不能绕过，强烈推荐

16.《心理学与生活》 (opens new window)

非常厚实的一部大砖头书，属于心理学入门书籍，内容非常扎实全面，几乎每一个章节都是一个主题内容，可以拆解研深，分量十足。通过阅读整书，能够让你全面和真正理解心理学是什么，严谨而有趣，从此摆脱伪心理学和鸡汤书，获得一个整观的心理学认识

17.《社会心理学》 (opens new window)

社会心理学经典著作，能够加深你对人与人之间，人与社会的认识，理解那些时常让人困惑的社会问题和道德情境，理解人的局限，理解人性

18.《乌合之众》 (opens new window)

解析群体心理的经典名著，对于群体行为的分析可谓鞭辟入里，入木三分，是社会心理学领域中最具影响力的著作，深刻思考群体行为并意欲有所作为的人都应该读读这本书，绝对不能绕过的经典之作

19.《伦理学与生活》 (opens new window)

让你对伦理学的概念不再是模模糊糊，真正理解道德与伦理，伦理与生活的关系，获得一些伦理学的常识和视角，当再次面对一些伦理问题和道德困境的时候不至于盲目和糊涂，真正进行抉择

20.《性学观止》 (opens new window)

获得基本的性学大观，而不是性技巧大观，摆脱盲目和无知

21.《哲学导论:综合原典阅读教程》 (opens new window)

综合原典，以八大哲学主题为核心进行理论阐述，时间上先后对应过渡，解读到位，接触过一点哲学史的人看是最合适的

22.《苏菲的世界》 (opens new window)

一部以小说的形式讲诉哲学史的书，哲学入门级的经典之作

23.《希腊·罗马文明》

书体很小，内容却足，讲述也不错，对于了解希腊文明和罗马辉煌很有帮助，基本面都到位了

24.《道德经》

通过‘道’与‘德’的方式讲述世间道理，体现了道家的思想观和处事态度，语言精妙优美，富含哲理

25.《娱乐至死》

对电视文化的‘控诉’，人的道德思想，人生观价值观都因媒介的改变而变化，媒介通过娱乐的方式展示一切，公众话语日渐以娱乐的方式出现，媒体以一种隐蔽却强大的暗示力量来“定义现实世界”，将文化变成一场娱乐至死的舞台。

26.《童年的消逝》

从过去‘童年’概念的形成，到如今‘童年’概念的逐渐‘消逝’，继《娱乐至死》后的又一次媒介控诉和批判，强调成年与童年的界限的消失，为的是警醒人们保留住‘童年’。

27.《爱的艺术》弗洛姆

真正让你明白爱是什么，爱意味着什么，爱需求什么——爱需要能力

28.《当尼采哭泣》

很特别的一本书，书中哲学家尼采帮助医生布雷尔解决人生困惑，布雷尔则帮助尼采解决心理困惑，最后双方都得到了升华，很喜欢亚隆文字里头透露出的洞察和通透。

29.《叔本华的治疗》

通过对菲利普性瘾症的治疗展示了团体治疗的风貌，挖掘出了潜藏在症状问题背后的防御机制和人生症结，此后迎来健壮清晰地人生，亚隆的书读完后，会感觉如沐更新，内心变得很有力量。

30.《存在主义心理治疗》

探讨了死亡，自由，存在的孤独和无意义这 4 个人类存在的核心问题，对它们的存在主义关怀和临床发挥能治愈患者，这里头体现出了亚隆渊博的知识背景和职业素养，存在主义治疗是一种哲学和心理学交融的心理治疗流派。

31.《害羞与社交焦虑症》

偏临床实操的一本心理学书籍，对于害羞和社交焦虑的有系统的治疗方案和训练指导，利用的是认知行为疗法，通过认知重塑，改变归因，观念应对和行为暴露，实现认知-生理-情绪-行为的正向矫正。

32.《人际关系心理学：学会相处之道》

一本被书名埋没的心理学书籍，不属于市面上那种纯社交经验类的书籍，而是以改善过后的认知行为疗法——人际取向疗法进行观念矫正和情绪调谐为主，以此来避免破坏性的人际交往和沟通，如果严格按里头的方法来，可以实现人格的正向塑造，对解决社交焦虑也很有用。

33.《驱散社交焦虑的阴霾》

一本内容结构很简单的书，但是麻雀虽小，却五脏俱全，关于社交焦虑的治疗有非常通俗易懂的策略与方法。

34.《社交焦虑的疏导整合疗法》

对精神分析理论和认知行为疗法的杂糅，作者的策略体系非常本土化，很符合国人患者的应用场景，唯一的问题是文笔一般，语词平白，读起来没有思维角力的感觉，认知体验上不过瘾。

35.《原生家庭》苏珊·福沃德

描述了不健康的家庭关系模式对子女的负面影响，这种影响造成了其性格结构的缺陷和消极的自我认知，阻碍了其成年后关系与生活的质量，书中提供了自我修复的方法。另外，这个系列的另外两本《依恋》和《执迷》也很棒，非常适合女性，讲述了依恋风格和原生家庭缺陷对女性在情感关系中的限制，以及未被知觉的情感操控和负性侵蚀，里头提供了应对方法。

36.《超越原生家庭》

介绍了家庭关系的运作原理，家庭关系会产生矛盾的原因，如何应对不合理的家庭模式，如何化解这种因它所致的对自身不好的影响。

37.《热锅上的家庭》

提出了家庭系统的概念，通过对一个濒临崩溃的家庭的治疗，展现了不合理因素是如何将家庭系统耗垮的，强调了原生家庭模式对子女婚姻的影响，是心理学中家庭疗法的案例体现。

38.《依恋的形成》

讲述了依恋研究的历史，强调了 3 岁前的母婴关系质量对儿童成长的影响——依恋风格，这种情感依恋会在成年后继续存在，在情感关系中得到体现，它影响着成年人的关系互动和质量。

39.《依恋与亲密关系》

这本书以依恋理论为基础，解释了亲密关系中的破坏性根源，提出以相互意识并满足对方依恋安全和情感联结为目标，进行情感关系的治愈，感觉非常有效，值得一读。

40.《情感勒索》

将一些将情感关系中的一些冷暴力，人际勒索和情感控制描述得非常全面和清楚，我们的生活中其实或多或少都存在部分特点。书中负性关系中的勒索者和被勒索者其实都存在缺乏安全感和缺爱，背后是原生家庭留下的缺陷和不安全型依恋，他们自己都没有意识到自己为何会如此，本书的意义在于意识，改变与应对。

41.《亲密关系》

与经验类阐述的书籍不同，这是一本基于社会心理学、沟通研究、家庭研究、认知心理学、发展心理学、演化心理学、社会学、传播学及家政学等学科的最新成果而成的一本书，里面的理论和观点都经过多次严谨的科学研究与验证。

42.《对伪心理学说不》

介绍了心理学是一门基于科学研究和实践的学科，驳斥了市面上很多缺乏研究支持，经不起研究，无法证伪的心理学，讽刺了那些经验类和鸡汤类的心理学，解释了大众对心理学认识上的误解，也揭示出了人类思维上存在的一些缺陷和漏洞。
