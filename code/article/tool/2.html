# Fiddler 抓取 https 方法

# 配置方法

菜单 —> menu —> tool —> https

  • 勾选"捕获 https 连接"

  • 勾选"捕获 https 流量"

  • 勾选"检查证书吊销状态"

# 常见问题

# 问题一

  • 解决 creation of the root certificate was not successful 报错

    • cd fiddler 安装目录

    • 键入如下命令

      makecert.exe -r -ss my -n "CN=DO_NOT_TRUST_FiddlerRoot, O=DO_NOT_TRUST, OU=Created by http://www.fiddler2.com" -sky signature -eku 1.3.6.1.5.5.7.3.1 -h 1 -cy authority -a sha1 -m 120 -b 09/05/2012

# 问题二

  • 解决 unable to configure windows to trust Fiddler Root certificate

    • 下载它 (opens new window)，进行安装

# 问题三

  • 网页提示 https 页面不安全

    1. menu —> tool —> https —> 动作 —> 导出证书

    2. 游览器 —> 高级 —> 证书 —> 导入 fiddler 证书
