# javascript 细节知识

  1. 字符串转数字：字符串 * 1。

  2. 其他类型转字符串：字符串" + 值。

  3. !!值，可以将值转换为布尔类型。

  4. !值，可以将值转换为布尔类型并取反。

  5. ""，null，undefined，0，NaN，在进行条件判断时，会被自动转换为false。

  6. Boolean，Number，String的new对象都可以通过valueOf()转换为对应值。

  7. for..in会自动过滤掉不存在的元素（比如数组设置 length 数）。

  8. 数组arr.length设大不会分配更多空间。

  9. 数组delete只会删除值，不会删除数组位。

  10. 对象字面量的"键"可为空字符串。

  11. Array.prototype.slice.call(arguments)转化速度比较慢。

  12. setTimeout/setInterval从第三个参数开始，全部都为函数参数项。

  13. eval只在被直接调用并且调用函数就是eval本身时，才在当前作用域中执行。

  14. typeof()中的括号代表的是分组操作符，而不是意味着 typeof 是作为函数使用。

  15. NaN——"不是数字"，属于特殊数字

  • 运算失败会得到 NaN

  • NaN 与任何数字运算结果都为 NaN

  1. （Infinity | -Infinity）——无穷数

  • 任何数除以 0 都为（Infinity|-Infinity）

  • Infinity 和-Infinity 的相互运算都为 NaN

  • Infinity 和-Infinity 与数字的常规运算都为（Infinity|-Infinity）

  1. 对象数组化
var obj = { age: 12 };

Array.prototype.push.call(obj, 1, 2);

console.log(obj);

// { 0: 1, 1: 2, age: 12, length: 2 }
